Entity Definition
YPM_PRICELIST_HDR
( MASTER Table )
This table stores the price list.



You can extend this table by adding columns but not by creating hang-off table(s). Please refer to the Customization Guide for detailed procedures and restrictions.

Tables should be accessed only through exposed APIs and Business Intelligence. Accessing them directly may result in issues during application of hot-fixes, service packs, and during upgrade to future releases of applications.

Columns
Column NameData TypeDescription
PRICELIST_HDR_KEYChar (24)The primary key for the YPM_PRICELIST_HDR table.  
PRICELIST_NAMEVarchar2 (90)Name of the price list. 
DESCRIPTIONVarchar2 (240)Description of the price list. 
START_DATE_ACTIVEDateTimeDate from which the price list is active. 
END_DATE_ACTIVEDateTimeAfter this date the price list is not active. 
CURRENCYVarchar2 (20)Prices in this price list are applicable in this currency.
A logical foreign key to the following table:
  • YFS_CURRENCY 
PRICING_STATUSVarchar2 (40)Indicates the status of the price list. The valid values are:
  • ACTIVE: Indicates that pricing entity is active.
  • INACTIVE: Indicates that pricing entity is not active. 
SELLER_ORGANIZATION_CODEChar (24)Seller who maintains this price list. Prices of items in the price list are defined by the seller.
A logical foreign key to the following table:
  • YFS_ORGANIZATION 
ORGANIZATION_CODEChar (24)Pricing organization to which the configuration belongs.
A logical foreign key to the following table:
  • YFS_ORGANIZATION 
INHERIT_FROM_PRICELIST_HDR_KEYChar (24)The pricelist header key of the price list from which this price list inherits its prices from. 
PERCENT_ADJUSTMENT_OVERRIDENumber (5,2)Percentage override valid only for a child price list. This adjustment is applied over the unit price obtained from the parent price list. 
ABSOLUTE_ADJUSTMENT_OVERRIDENumber (19,6)Absolute override valid only for a child price list. This adjustment is applied over the unit price obtained from the parent price list. 
MARKED_FOR_LIST_PRICEChar (1)Indicates this price list is marked for listing prices. Applicable only for master price list. 
LOCKIDNumber (5,0)An integer used to track and prevent concurrent modifications.

CREATEPROGIDVarchar2 (40)The program that created this record.
