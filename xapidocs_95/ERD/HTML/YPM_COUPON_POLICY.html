Entity Definition
YPM_COUPON_POLICY
( CONFIGURATION Table )
table maintains coupon policy information for generated coupon codes



This table is not extensible.

Tables should be accessed only through exposed APIs and Business Intelligence. Accessing them directly may result in issues during application of hot-fixes, service packs, and during upgrade to future releases of applications.

Columns
Column NameData TypeDescription
COUPON_POLICY_KEYChar (24)The primary key for the YPM_COUPON_POLICY table.  
ORGANIZATION_CODEChar (24)Organization to which the coupon policy belongs to 
NAMEVarchar2 (40)Name of the character set definition 
COUPON_ID_LENGTHNumber (2,0)The number of characters, starting at the beginning, of the coupon id to use in the coupon code string. 
COUPON_ID_STARTING_INDEXNumber (2,0)the starting index in the coupon code string for the coupon id. If the coupon id appears as a prefix in the coupon code then the starting index will be 0. If the coupon Id should appear as a suffix then the starting index should be mentioned as -1 
PAD_CHARChar (1)if the give coupon id is shorter than the coupon_id_length, the remaining spaces will be filled with this character. 
START_DATE_ACTIVEDateTimedate and time when the policy becomes active. no two policies can be active at the same time. 
END_DATE_ACTIVEDateTimedate and time when the policy becomes inactive. no two policies can be active at the same time.

  • ACTIVE: Indicates that pricing entity is active.
  • INACTIVE: Indicates that pricing entity is not active.
