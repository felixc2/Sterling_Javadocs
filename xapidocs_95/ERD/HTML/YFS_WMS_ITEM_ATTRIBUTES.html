YFS_WMS_ITEM_ATTRIBUTES
( MASTER Table )
Stores WMS specific product information.



You can extend this table by adding columns and by creating hang-off table(s). Please refer to the Customization Guide for detailed procedures and restrictions.

Tables should be accessed only through exposed APIs and Business Intelligence. Accessing them directly may result in issues during application of hot-fixes, service packs, and during upgrade to future releases of applications.

DEFORMABLEChar (1)Indicates whether this item can be folded/stuffed/squeezed. 
PACKED_HEIGHTNumber (14,4)Height of the item when packed inside a container. The unit of measure for this is same as that of dimension UOM for unit height. 
PACKED_LENGTHNumber (14,4)Length of the item when packed inside a container. The unit of measure for this is same as that of dimension UOM for unit length. 
PACKED_WIDTHNumber (14,4)Width of the item when packed inside a container. The unit of measure for this is same as that of dimension UOM for unit width. 
STACKED_HEIGHT_INCNumber (14,4)Increase in height of the item when stacked. The unit of measure for this is same as that of dimension UOM for unit height. 
CONSTRAINED_BY_HEIGHTChar (1)Indicates whether the item orientation when packed inside the container is constrained by height, for ex: up-right or non-tippable items like paints, oil cans etc. 
PACK_BY_VOLUMEChar (1)Indicates whether the item dimensions should be tracked after this item is placed inside the container. If true, only the volume and weight are tracked after performing a dimension check against the container to be packed. 
LOCKIDNumber (5,0)An integer used to track and prevent concurrent modifications.
