Entity Definition
YFS_INVENTORY_SUPPLY_TYPE
( CONFIGURATION Table )
Configuration table. Stores valid supply types.



This table is not extensible.

Tables should be accessed only through exposed APIs and Business Intelligence. Accessing them directly may result in issues during application of hot-fixes, service packs, and during upgrade to future releases of applications.

Columns
Column NameData TypeDescription
INVENTORY_SUPPLY_TYPE_KEYChar (24)The primary key for the YFS_INVENTORY_SUPPLY_TYPE table.  
SUPPLY_TYPEVarchar2 (40)A Supply type code. 
DESCRIPTIONVarchar2 (100)Description of a supply type. 
COSTING_REQUIREDChar (1)Set to 'Y' if this suppy type is to tracked for cost purposes. 
RETAIN_REFERENCEChar (1)Set to 'Y' if supply type requires reference # level tracking. If set to 'N', reference # are not kept in SUPPLY table. 
ONHAND_SUPPLYChar (1)Set to 'Y' if this supply type if considered as on-hand. ETA in SUPPLY table is set to low date to indicate immediate availability. 
SYSTEM_DEFINEDChar (1)Set to 'Y' if this supply type is a factory setting at product release time. 
TRIGGER_IBAChar (1)This flag indicates whether the supply type can trigger the Item-Based Allocation process when supply changes occur. Valid values: Y/N. Default value: N. This is available to be configured only for non on-hand supply types. 
USE_DEMAND_TYPE_FOR_IBAChar (1)This flag indicates that the demand types, which consider this supply type, should be considered for Item Based Allocation. Valid values: Y/N. Default value: N. 
IS_AUDIT_ENABLEDChar (1)This flag indicates that inventory audits for supply change should be logged for this supply type. Valid values: Y/N. Default value: N.
