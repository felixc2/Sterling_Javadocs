Entity Definition
YFS_WORK_AREA
This table stores work area definition for each user group. Work area is the set of locations (zone and aisle combination) within which a set of users work. System should suggest tasks originating and terminating only in these areas.

This table cannot be extended.

Sterling tables should be accessed only through Sterling APIs and Sterling Analytics. Accessing them directly may result in issues during application of hot-fixes, service packs, and during upgrade to future releases of Sterling.
