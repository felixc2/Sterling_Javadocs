Entity Definition
YFS_RECEIVING_DISCR_REASON
( CONFIGURATION Table )
Configuration table. This table stores the master setup for the discrepancy reason codes. The codes cannot be shared between document types. However, an enterprise can inherit this setup from anther enterprise.



This table is not extensible.

Tables should be accessed only through exposed APIs and Business Intelligence. Accessing them directly may result in issues during application of hot-fixes, service packs, and during upgrade to future releases of applications.
