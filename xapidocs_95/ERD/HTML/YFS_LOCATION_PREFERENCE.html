Entity Definition
YFS_LOCATION_PREFERENCE
( CONFIGURATION Table )
For an activity, the preference list of zones where that activity may be performed is configured by priority.



This table is not extensible.

Tables should be accessed only through exposed APIs and Business Intelligence. Accessing them directly may result in issues during application of hot-fixes, service packs, and during upgrade to future releases of applications.

LOCATION_STORAGE_CODEVarchar2 (40)Ths field is an attribute of the location that allows the warehouse to store items, which has the same storage profile as that of the location. For example, hazardous inflammable items need locations close to fire extinguishers. In this case, the locations will be marked as having a storage code, which is suitable for storing 'Inflammable item'. This ensures that only inflammable items get to these locations. A list of zones in the order of priority will be used for putaway after filtering out locations that honor the storage code. 
PRIORITYNumber (10,0)This field indicates the zone priority where the product must be putaway based upon the business criteria.
