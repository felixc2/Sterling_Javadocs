Entity Definition
YFS_WORK_TYPE_DETERMINATION
This table stores parameters that are used to determine work type for a task.

This table cannot be extended.

Sterling tables should be accessed only through Sterling APIs and Sterling Analytics. Accessing them directly may result in issues during application of hot-fixes, service packs, and during upgrade to future releases of Sterling.
