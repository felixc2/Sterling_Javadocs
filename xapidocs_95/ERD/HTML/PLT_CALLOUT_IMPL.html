Entity Definition
PLT_CALLOUT_IMPL
( CONFIGURATION Table )
This table stores the callout implementation details for all callouts



This table is not extensible.

Tables should be accessed only through exposed APIs and Business Intelligence. Accessing them directly may result in issues during application of hot-fixes, service packs, and during upgrade to future releases of applications.

SEQUENCE_NONumber (5,0)The sequence number for this callout implementation. The default value of this field is 0. If a negative value is passed, this will be set to 0.

IS_SUSPENDEDChar (1)This field is set to Y if the execution of the CallOut implementation has resulted in a TimeOut.
