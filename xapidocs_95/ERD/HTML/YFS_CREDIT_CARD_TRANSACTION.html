Entity Definition
YFS_CREDIT_CARD_TRANSACTION
( TRANSACTION Table )
This table stores detailed information about authorizations and charges made against all supported payment types. Typically, this information would be returned by credit card interface programs such as CyberSource or PaymentTek.

Records purged from this table are archived into YFS_CREDIT_CARD_TRANSACTION_H

This table is not extensible.

Tables should be accessed only through exposed APIs and Business Intelligence. Accessing them directly may result in issues during application of hot-fixes, service packs, and during upgrade to future releases of applications.
