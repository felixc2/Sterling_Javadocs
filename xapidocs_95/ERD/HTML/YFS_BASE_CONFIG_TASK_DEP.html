Entity Definition
YFS_BASE_CONFIG_TASK_DEP
( CONFIGURATION Table )
This table shows the dependencies between tasks. There will be many to many relationships between tasks. A task can have multiple predecessors on which it depends. Additionally, multiple tasks may depend on a single task.



This table is not extensible.

Tables should be accessed only through exposed APIs and Business Intelligence. Accessing them directly may result in issues during application of hot-fixes, service packs, and during upgrade to future releases of applications.
