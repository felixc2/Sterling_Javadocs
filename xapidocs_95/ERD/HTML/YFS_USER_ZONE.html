Entity Definition
YFS_USER_ZONE
( MASTER Table )
This table stores the work area definition for each user group. Work area is the set of locations, consisting of the zone and aisle combination within which a set of users work. The system suggests tasks originating and terminating only in these areas.



This table is not extensible.

Tables should be accessed only through exposed APIs and Business Intelligence. Accessing them directly may result in issues during application of hot-fixes, service packs, and during upgrade to future releases of applications.
