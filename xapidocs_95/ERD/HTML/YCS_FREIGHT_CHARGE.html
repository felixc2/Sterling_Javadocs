Entity Definition
YCS_FREIGHT_CHARGE
( CONFIGURATION Table )
This table stores the freight charges.



This table is not extensible.

Tables should be accessed only through exposed APIs and Business Intelligence. Accessing them directly may result in issues during application of hot-fixes, service packs, and during upgrade to future releases of applications.

Columns
Column NameData TypeDescription
FREIGHT_CHARGE_KEYChar (24)The primary key for the YCS_FREIGHT_CHARGE table.  
CARRIERVarchar2 (20)Indicates the carrier to which the package is manifested. 
SERVICEChar (2)Indicates the service to which package is manifested. 
ACCOUNT_TYPEChar (3)Indicates the RE or CO. 
WEIGHTNumber (22,4)Indicates the weight of the package. 
ZONEChar (5)The zone as given by the Carrier. 
MIN_CHARGENumber (22,4)The basic charge specified by the Carrier. 
FACTOR_CHARGENumber (22,4)This field is used in basic charge computation.If MIN_CHARGE > (FACTOR_CHARGE * Weight) then MIN_CHARGE is considered as the basic charge. Else,(FACTOR_CHARGE * Weight) is considered as the basic charge.
