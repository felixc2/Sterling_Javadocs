Entity Definition
YFS_ENTITY_ATTRIBUTE_DOMAIN
( CONFIGURATION Table )
This table stores the Attribute domain used by an entity TableName and DocumentType.



This table is not extensible.

Tables should be accessed only through exposed APIs and Business Intelligence. Accessing them directly may result in issues during application of hot-fixes, service packs, and during upgrade to future releases of applications.

Columns
Column NameData TypeDescription
ENTITY_ATTRIBUTE_DOMAIN_KEYChar (24)The primary key for the YFS_ENTITY_ATTRIBUTE_DOMAIN table.  
ORGANIZATION_CODEChar (24)The organization that maintains the YFS_ATTRIBUTE_DOMAIN record given in ATTRIBUTE_DOMAIN_ID column.
A logical foreign key to the following tables:
  • YFS_ATTRIBUTE_DOMAIN (in combination with the ATTRIBUTE_DOMAIN_ID column)
  • YFS_ORGANIZATION 
ATTRIBUTE_DOMAIN_IDChar (40)The primary identifier of the record in YFS_ATTRIBUTE_DOMAIN table.
A logical foreign key to the following table:
  • YFS_ATTRIBUTE_DOMAIN (in combination with the ORGANIZATION_CODE column) 
DOCUMENT_TYPEVarchar2 (40)The DocumentType with which the AttributeDomain defined in column ATTRIBUTE_DOMAIN_ID must be associated. 
TABLE_NAMEVarchar2 (40)The TableName with which the AttributeDomain defined in column ATTRIBUTE_DOMAIN_ID must be associated. The allowed values are YFS_ORDER_HEADER, YFS_ORDER_LINE AND YFS_OPPORTUNITY 
CREATETSDateTimeThe timestamp for the creation of this record. 
MODIFYTSDateTimeThe timestamp for the last modification of this record. 
CREATEUSERIDVarchar2 (40)The user who created this record. 
MODIFYUSERIDVarchar2 (40)The user who last modified this record. 
CREATEPROGIDVarchar2 (40)The program that created this record. 
MODIFYPROGIDVarchar2 (40)The program that modified this record. 
LOCKIDNumber (5,0)An integer used to track and prevent concurrent modifications.
