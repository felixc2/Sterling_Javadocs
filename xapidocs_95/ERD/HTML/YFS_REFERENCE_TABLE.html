Entity Definition
YFS_REFERENCE_TABLE
( TRANSACTION Table )
This table stores dynamic order references that can be passed to the createOrder API and returned by the getOrderDetails API. Database extensibility of the order tables should be used instead of this table to store implementation specific order fields.

Records purged from this table are archived into YFS_REFERENCE_TABLE_H

This table is not extensible.

Tables should be accessed only through exposed APIs and Business Intelligence. Accessing them directly may result in issues during application of hot-fixes, service packs, and during upgrade to future releases of applications.
