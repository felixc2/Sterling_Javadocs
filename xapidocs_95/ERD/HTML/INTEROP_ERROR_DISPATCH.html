Entity Definition
INTEROP_ERROR_DISPATCH
( TRANSACTION Table )
This table is populated when a error is marked for reprocessing. When an IntegrationServer is started the server peridocially (every minute) looks at this table to see if there are any records that can be processed for the flow or sub flow (Service/RuntimeID) combination.



This table is not extensible.

Tables should be accessed only through exposed APIs and Business Intelligence. Accessing them directly may result in issues during application of hot-fixes, service packs, and during upgrade to future releases of applications.
