Entity Definition
YFS_ORDER_KIT_LINE
( TRANSACTION Table )
This table stores information about kitted component items. A kit is a collection of items that form a unit. For example, a home theatre system is a unit consisting of component items such as a DVD player, TV, CD player, etc.

Records purged from this table are archived into YFS_ORDER_KIT_LINE_H

You can extend this table by adding columns but not by creating hang-off table(s). Please refer to the Customization Guide for detailed procedures and restrictions.

Tables should be accessed only through exposed APIs and Business Intelligence. Accessing them directly may result in issues during application of hot-fixes, service packs, and during upgrade to future releases of applications.
