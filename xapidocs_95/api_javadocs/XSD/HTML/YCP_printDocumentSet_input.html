PrintDocument Associated With Table: YFS_PRINT_DOCUMENT
Specifies the PrintDocumentId and LabelFormats of the documents that have to be printed before and after processing the documents specified in the child PrintDocuments node. A PrintDocument node can consist of one PrintDocuments node, which in turn can consist of multiple PrintDocument nodes. Hence, PrintDocument node is recursive in nature.
