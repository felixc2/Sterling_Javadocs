CopyFromPricelistHeader Associated With Table: YPM_PRICELIST_HDR
The price list to copy from. Specify PricelistHeaderKey or PricelistName to identify a price list. If this price list is copied to a new price list, all attributes copied if not specified for the new price list. If this price list is copied to an existing price list, attributes are not copied. InheritFromPricelistHeaderKey, AbsoluteAdjustmentOverride and PercentAdjustmentOverride are copied only if the new price list has the same currency.

Validation ConditionError
Not able to find a price list for the passed price list identifiers.
YPM10016Invalid price list identifiers.
