AdditionalAttribute Associated With Table: YFS_ADDITIONAL_ATTRIBUTE
Stores dynamic attributes attached with an entity. Currently used to store additional attributes for item and category entities.

fraction digits: 4) 		yfs_additional_attribute.double_value

IntegerValuenonNegativeInteger (total number of digits: 20) yfs_additional_attribute.integer_value
The value of the assigned attribute whose data type is 'INTEGER'.
IsInheritedflag (maximum length: 1) yfs_additional_attribute.is_inherited
This field indicates whether the assignment of attribute value as additional attribute was based on inheritance. For ParentTable='YFS_CATEGORY', a value of 'Y' indicates that the assignment is based on inheritance of the value assigned to the upper level category. For ParentTable='YFS_ITEM', a value of 'Y' indicates that the assignment is based on inheritance of the value assigned to the category to which the item belongs.
Namestring (maximum length: 40) yfs_additional_attribute.name
Name of the dynamic attribute.
Operationstring (maximum length: 40) .
Operation to perform
Valid Values:
Manage - Creates a new record if it does not exist, or modifies an existing record if it does.
Create - Creates a new record.
Modify - Change values in an existing record.
Delete - Deletes an existing record.
The default behavior is Manage .
