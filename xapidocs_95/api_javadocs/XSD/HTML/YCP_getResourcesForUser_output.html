Redirectorflag (maximum length: 1) yfs_resource.redirector
When a view is marked as a redirector, then standard layout is not used for that view. Once the APIs are invoked, control is directly transfered to the JSP of the view. If a view is marked as redirector, then JSP must be specified. An example use for this is when dynamically determining a view for an action. If an action is configured to go to a view that is marked as a redirector, then the redirector can dynamically determine the actual view to go to. Thus, we get the feature of having Action that can go to a dynamically determined view.
