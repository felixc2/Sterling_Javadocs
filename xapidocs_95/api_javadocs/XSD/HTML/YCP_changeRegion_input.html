Region Associated With Table: YFS_REGION
This table contains information about the region definition. A region schema key has to be given to indicate the schema to which the region is associated.

Validation ConditionError
Either RegionKey or {RegionName and ParentRegionKey and RegionSchemaKey} or {RegionName and ParentRegionKey and RegionSchemaName and OrganizationCode} should be passed to identify the region. ParentRegionKey can be blank if no parent region exists for a region. If not passed:
YCP0158Invalid region passed
AttributeTypeRequiredTable.Column
Actionstring.
Indicates the action that can be perfomed on the region.
Default: MODIFY
Valid Values:
DELETE: Deletes a region. No Output XML is generated in this case.
MODIFY: Changes a region.

CanAggregateflag (maximum length: 1) yfs_region.can_aggregate
A flag that indicates whether this region can be a parent of other regions. If CanAggregate is Y, the region cannot have zipcode ranges defined. If CanAggregate is N, this region cannot become the parent of any other region. Any modification to this flag is validated by checking the region's zipcode ranges as well as child regions.

Countrystring.

ParentRegionKeykey (maximum length: 24) yfs_region.parent_region_key
The parent region associated with this region.
RegionDescriptionstring (maximum length: 100) yfs_region.region_description
The description of this region.
RegionKeykey (maximum length: 24) yfs_region.region_key
The primary key for the YFS_REGION table.
RegionLevelNamestring (maximum length: 40) yfs_region.region_level_name
The Region Level for the region. If RegionLevelName is changed, the API checks for associations of the new level with the parent Region's level and the child Regions' levels.

RegionNamestring (maximum length: 40) yfs_region.region_name
The unique name of this region.
ServicesEntireCountryflag (maximum length: 1) .
The flag indicates if this region applies to the entire country.

RegionSchemaNamestring (maximum length: 40) yfs_region_schema.region_schema_name

ZipCodeRange Associated With Table: YFS_REGION_DETAIL

Validation ConditionError
The ToZip field must be greater than or equal to the FromZip field. If not:
YCP0128FromZip cannot be greater than ToZip.
The length of the ToZip field must be equal to the length of the FromZip field. If not:
YCP0127Zipcode length mismatch.
AttributeTypeRequiredTable.Column
Actionstring.
If Action is passed with a value of REMOVE, the zipcode range is deleted for the region. If Action is passed as blank or not passed, the zipcode range is created for that region.

Countrystring (maximum length: 40) yfs_region_detail.country
The country for which the region is created.

Validation ConditionError
If not passed:
YCP0129Invalid country.

FromZipstring (maximum length: 35) yfs_region_detail.from_zip
The zip code from which the range starts.
ToZipstring (maximum length: 35) yfs_region_detail.to_zip
The zip code where the range ends.
