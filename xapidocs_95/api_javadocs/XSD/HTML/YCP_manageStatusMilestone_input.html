ElementsStatusMilestone
StatusMilestone Associated With Table: YFS_STATUS_MILESTONE
Stores milestones attached to a status. StatusMilestoneKey or a unique combination of ProcessTypeKey, Status, DateTypeId and MilestoneLevel can be specified to identify a StatusMilestone.

Validation ConditionError
If ProcessTypeKey, Status, DateTypeId or MilestoneLevel is not passed in Create mode.
YCP0296ProcessTypeKey, Status, MilestoneLevel and DateTypeId are mandatory.
If existing StatusMilestoneKey or a unique combination of ProcessTypeKey, Status, DateTypeId and MilestoneLevel is passed in Create mode
YFC0001Record already exists in the database.
If record does not exist in the database
YFC0002Record does not exist in the database.
If an invalid combination of Status and ProcessTypeKey is passed
YCP0298Combination of Status and ProcessTypeKey does not exist in YFS_Status table.
If an invalid combination of DateTypeId, ProcessTypeKey and OwnerKey is passed
YCP0299Combination of DateTypeId, ProcessTypeKey and OwnerKey does not exist in YFS_Date_Type table.
