Question Associated With Table: YFS_QUESTION
This table contains information questions for question types defined by an organization.
AttributeTypeRequiredTable.Column
AnswerValuestring (maximum length: 1000) .
The answer provided for this question from the answer set, if applicable.
DataTypestring (maximum length: 20) yfs_question.data_type
Indicates what type of data type is expected for the answer. Valid values are BOOLEAN, DECIMAL, INTEGER, TEXT.
DisplayControlTypestring (maximum length: 20) yfs_question.display_control_type
Indicates what type of display control to show for the user interface when supplying an answer. Valid values are CHECK_BOX, DROPDOWN, RADIO_BUTTON, TEXT_BOX, TEXT_FIELD.
IsGroupflag (maximum length: 1) yfs_question.is_group
Indicates if the question is a grouping for a set of questions.
OrganizationCodestring (maximum length: 24) yfs_question.organization_code
The organization that defined the question.
ParentAnswerOptionKeykey (maximum length: 24) yfs_question.parent_answer_option_key
Answer Option Key of the answer option that is required to be selected in order to ask this question.
ParentQuestionKeykey (maximum length: 24) yfs_question.parent_question_key
Question Key of the question group to which this question belongs.
QuestionIDstring (maximum length: 20) yfs_question.question_id
Identifier for the question.
QuestionKeykey (maximum length: 24) yfs_question.question_key
Primary key of question.
QuestionTextstring (maximum length: 200) yfs_question.question_text
The text to display when posing the question.
QuestionTypestring (maximum length: 20) yfs_question.question_type
The question type.
SequenceNononNegativeInteger (total number of digits: 5) yfs_question.sequence_no
Specifies for the question hierarchy level of this question, what sequence to display this question in.
