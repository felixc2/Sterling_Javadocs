ConfigureBatchBy
This ConfigureBatchBy element is used to create new batches based on the attributes passed. This element is considered only when GetNewBatches=Y. Having many attributes in ConfigureBatchBy element may have performance implications when large number of shipment lines are to be included in the batches. If no ConfigureBatchBy element is passed, then default attributes used are DepartmentCode of ShipmentLine entity and EnterpriseCode of Shipment entity.
