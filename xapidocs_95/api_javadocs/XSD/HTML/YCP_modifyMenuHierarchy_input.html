Menu Associated With Table: YFS_MENU
Stores information on how a menu needs to be organized within an application.
AttributeTypeRequiredTable.Column
DisplayModestring (maximum length: 40) yfs_menu.display_mode
Display mode in which a menu needs to be displayed. For example, Dialog, Docked, or HelpMenu.
Iconstring (maximum length: 255) yfs_menu.icon
An Icon associated with a menu item.
MenuDescriptionstring (maximum length: 100) yfs_menu.menu_description
Description of a menu item.
MenuKeykey (maximum length: 24) Yesyfs_menu.menu_key
Primary key attribute of YFS_MENU.
MenuSeqnonNegativeInteger (total number of digits: 5) yfs_menu.menu_seq
A sequence number used for sorting child menu items below a particular menu item.
MenuTypestring (maximum length: 10) yfs_menu.menu_type
Type of a menu. MENU/RESOURCE. RESOURCE typed menus are leaf nodes in the menu hierarchy that have an associated resource to access. Menu Items of type MENU are just place holders creating an organized menu tree.
ParentMenuKeykey (maximum length: 24) yfs_menu.parent_menu_key
Parent menu item to which a menu item belongs.
ResourceKeykey (maximum length: 24) yfs_menu.resource_key
A resource associated with a menu item. This is populated only for leaf menu items.
