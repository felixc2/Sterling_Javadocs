InheritFromPricelistHeader
InheritFromPricelistHeader Associated With Table: YPM_PRICELIST_HDR
The master price list from which this price list inherits it's prices. A master price list is one that defines the prices of items rather than inheriting item prices from a different price list.

Validation ConditionError
If the specified master price list is already inherited from another price list. Only one level of inheritance is supported.
YPM10009Cannot inherit from a child price list.
The master price list is in a different currency of that of child.
YPM10008Child and inherit from price list cannot have different currency.
