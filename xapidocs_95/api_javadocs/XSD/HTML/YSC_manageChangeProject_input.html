ChangeProject Associated With Table: YFS_CHANGE_PROJECT
This table stores Change Project definitions. A Change Project provides the mechanism to consolidate database changes under a single unit. A Change Project can be approved on one system so that the database changes associated with it can be sent to another system. A Change Project is a collection of Change Requests. It is also the unit which a valid user can approve or reject. All data changed as part of a Change Project will be logged under the same Change Project.
