ConsolidationTemplate
If this element is passed then all attributes specified in the child Inbox element will be used to find an exception for consolidation. If this element is not passed, then the default consolidation template will be used instead. Note that consolidation will only occur if the Consolidate flag in the parent Inbox element is passed as Y.
  • An Extn element may be passed as a child of this Inbox element to indicate extended fields to be used for consolidation.
  • If both the Extn and Inbox elements are passed, but neither one has any attributes, then consolidation will take place on all extended fields [and none of the standard ones].
