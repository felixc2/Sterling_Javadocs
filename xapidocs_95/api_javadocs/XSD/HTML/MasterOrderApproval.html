OrderApprovalDetail Associated With Table: YFS_ORDER_APPROVAL_DTL
This tables stores order approval details which get created either when an quote is created/changed or when a quote is sent for approval. These records are created based on approval plan and order approval rules. An order approval detail without an order approval record indicates that the quote has not been sent for approval yet. An order approval detail associated to an order approval indicates what a user would be approving while approving the associated order approval.
