Award Associated With Table: YFS_PROMOTION_AWARD
This table stores information regarding awards that are created out of the promotions or information regarding pricing rules applied on the order. A single promotion or pricing rule can result in multiple awards.
AttributeTypeRequiredTable.Column
Actionstring (maximum length: 40) .
Action to perform on this Award.
Default: Null - if not passed awards will be added if not exist or modified.
Valid Values:
REMOVE - Removes the award.

AwardAmountdecimal (total number of digits: 15
fraction digits: 2) 		yfs_promotion_award.award_amount
The amount of discount or charge that was applied as a part of this award.
AwardAppliedflag (maximum length: 1) yfs_promotion_award.award_applied
A flag indicating if the award is applied. When ValidateReferences=Y the value can be 'P' meaning the award was partially applied
AwardIdstring (maximum length: 120) yfs_promotion_award.award_id
A unique identifier for the award. If pricing is maintained within the system, this will be the PricingRuleName of the pricing rule that created this award.
AwardKeykey (maximum length: 24) yfs_promotion_award.promotion_award_key
The primary key of the YFS_PROMOTION_AWARD table.
AwardTypestring (maximum length: 100) yfs_promotion_award.award_type
The type of award. If pricing is maintained within the system, this will be the RuleCategory of the pricing rule that created this award.
ChargeCategorystring (maximum length: 20) yfs_promotion_award.charge_category
The charge category of the LineCharge or HeaderCharge for which this award was created.
ChargeNamestring (maximum length: 40) yfs_promotion_award.charge_name
The charge name of the LineCharge or HeaderCharge for which this award was created.
DenialReasonstring (maximum length: 100) yfs_promotion_award.denial_reason
If the award is denied, the reason for the denial.
Descriptionstring (maximum length: 2000) yfs_promotion_award.description
A description for the award. If pricing is maintained within the system, this will be the Description of the pricing rule that created this award.
PosReasonCodestring (maximum length: 100) yfs_promotion_award.pos_reason_code
Reason this promotion was applied.
PromotionIdstring (maximum length: 100) yfs_promotion_award.promotion_id
This identifies the promotion that is added to this order.
PromotionKeykey (maximum length: 24) yfs_promotion_award.promotion_key
The promotion this award belongs to.
