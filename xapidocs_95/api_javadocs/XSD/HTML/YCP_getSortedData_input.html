XMLPathstringYes.
Path of the parent output XML element whose children will be sorted. For example, the User elements in the following XML will be sorted when an XMLPath of /UserList is given.
<Organization>
    <UserList>
        <User Loginid="User1"/>
        <User Loginid="User2"/>
    </UserList>
</Organization>
To select the root element, use a single slash (/)
