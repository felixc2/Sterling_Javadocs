ShortDescriptionstring (maximum length: 100) yfs_attr_allowed_value.short_description
The short description of the allowed value.
Valuestring (maximum length: 200) yfs_attr_allowed_value.value
The allowed value of the attribute that the AttributeAllowedValue represents. For an attribute of INTEGER or DECIMAL Datatype, this value may be a list of values or range of values. Each value in the list may be separated by a ';'. A range must be specified by two values separated by a '-'.

Validation ConditionError
If the Value attribute is a range, then the two numbers in the range must be separated by a '-'.
YSC84_0007Range Not Defined Correctly.
If the Value attribute is a range, then the first value must be the smaller value and second value must be the greater one.
YSC84_0002Min Value Cannot Be Greater Than Max Value In A Range.
If the Attribute is configured with AllowMultipleValues set to 'N' and this AttributeAllowedValue has IsDefaultValue set to 'Y', then the Value must not be a range.
YSC84_0017Cannot Define Multiple Default Values For This Attriute.
If the AllowedValueLength is configured for this AttributeAllowedValue record, then the length of the Value string must not exceed the AllowedValueLength.
YSC84_0012Length Of the Value String Exceeds The Allowed Value Length For The Attribute.
The Value string must contain valid Decimal numbers.
YSC84_0010Defined Value Not Valid For Decimal Data Type.
The Value string must contain valid integers.
YSC84_0018Defined Value Not Valid For Integer Data Type.
