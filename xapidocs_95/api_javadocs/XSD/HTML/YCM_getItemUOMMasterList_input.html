ElementsItemUOMMaster
ItemUOMMaster Associated With Table: YFS_ITEM_UOM_MASTER
This table stores all valid UOMs that are valid for a catalog. All operational, selling and inventory UOMs are maintained in this table.
AttributeTypeRequiredTable.Column
CallingOrganizationCodestring (maximum length: 24) .
If passed, a list of master UOMs is returned for the catalog organization of the calling organization. If OrganizationCode is also passed, it is used as a filter; the returned list will contain any master UOMs with the OrganizationCode.
