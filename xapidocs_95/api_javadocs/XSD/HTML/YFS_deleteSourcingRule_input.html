ElementsSourcingRuleHeader
SourcingRuleHeader Associated With Table: YFS_SOURCING_RULE_HDR
This table stores information about sourcing rule headers which provide selection criteria for sourcing, procurement and service nodes for each organization.
AttributeTypeRequiredTable.Column
FulfillmentTypestring (maximum length: 40) yfs_sourcing_rule_hdr.fulfillment_type
The fulfillment type of this sourcing rule. This field can be matched with the corresponding field in the the YFS_ORDER_LINE table and the appropriate sourcing rule can be selected during scheduling.
ItemClassificationstring (maximum length: 40) yfs_sourcing_rule_hdr.item_classification
This field indicates the item classification associated with the <b>Sourcing</b> classification purpose. This field is only valid in the context of product sourcing for shipping. A blank item classification specifies all classifications.
ItemClassification2string (maximum length: 40) yfs_sourcing_rule_hdr.item_classification2
This field indicates the item classification associated with the <b>Sourcing2</b> classification purpose. This field is only valid in the context of product sourcing for shipping. A blank item classification specifies all classifications.
ItemClassification3string (maximum length: 40) yfs_sourcing_rule_hdr.item_classification3
This field indicates the item classification associated with the <b>Sourcing3</b> classification purpose. This field is only valid in the context of product sourcing for shipping. A blank item classification specifies all classifications.
ItemGroupCodestring (maximum length: 20) yfs_sourcing_rule_hdr.item_group_code
This field incidates what kind of item this is.
Allowed ValueDescription
PROD Indicates that the item is a product item. 
PS Indicates that the item is a provided service item. 
DS Indicates that the item is a delivery service item. 
SS Indicates that the item is a special service item. 

ItemIDstring (maximum length: 40) yfs_sourcing_rule_hdr.item_id
The identifier of the item for which sourcing is setup. This field is only valid only for product sourcing for shipping.
NodeTypeIDstring (maximum length: 40) yfs_sourcing_rule_hdr.node_type_id
Indicates node type id of the receiving node associated to the sourcing rule.
OrderSourcingClassificationstring (maximum length: 40) yfs_sourcing_rule_hdr.order_sourcing_classification
The order sourcing classification used for this sourcing rule. This field can be matched with the corresponding field in the YFS_ORDER_HEADER table and the appropriate sourcing rule can be selected during scheduling.
OrganizationCodestring (maximum length: 24) yfs_sourcing_rule_hdr.organization_code
The organization that is associated with this sourcing rule.
This attribute corresponds to the OrganizationCode Fact.
Purposestring (maximum length: 40) yfs_sourcing_rule_hdr.purpose
The purpose of this sourcing rule.
Allowed ValueDescription
SOURCING Indicates that this rule is a sourcing rule. 
PROCUREMENT Indicates that this rule is a procurement rule. 

RegionKeykey (maximum length: 24) yfs_sourcing_rule_hdr.region_key
The region that is associated with this sourcing rule.
SellerOrganizationCodestring (maximum length: 24) yfs_sourcing_rule_hdr.seller_organization_code
The seller organization that is associated with this sourcing rule.
This attribute corresponds to the OrganizationCode Fact.
SourcingRuleHeaderKeykey (maximum length: 24) yfs_sourcing_rule_hdr.sourcing_rule_hdr_key
The primary key for the YFS_SOURCING_RULE_HDR table.
ToNodeKeykey (maximum length: 24) yfs_sourcing_rule_hdr.to_node_key
The ship node that is associated with this sourcing rule.
This attribute corresponds to the OrganizationCode Fact.
