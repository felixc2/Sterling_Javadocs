AdditionalAttribute
AdditionalAttribute Associated With Table: YFS_ADDITIONAL_ATTRIBUTE
This entity is used to store values for an attribute assigned to a category or item. This entity is also used to store dynamic attributes attached with an entity. Currently used to store additional attributes for item and category entities.

Validation ConditionError
If this element is passed, Name attribute cannot be blank.
YFS10364Mandatory Parameters for the Operation are missing.
If this element is passed, either Value or IntegerValue or DoubleValue must be passed.
YFS10364Mandatory Parameters for the Operation are missing.
An AdditionalAttribute record is not allowed for an Attribute record that has an AttributeReference associated to it.
YSC84_0013Cannot Assign Value To Reference Attribute.
If the associated Attribute record is configured with AllowMultipleValues as 'N', then only one AdditionalAttribute record is allowed for that Attribute for the associated item or category of this AdditionalAttribute record.
YSC84_0011Cannot Define Multiple Values For This Attribute.
If the associated Attribute record is of DataType 'TEXT' and is configured with an AllowedValueLength and the flag IsAllowedValueDefined is not set to 'Y', then the length of this Value attribute may not exceed the defined AllowedValueLength.
YSC84_0012Length Of Assigned Value Cannot Exceed Allowed Value Length Of The Attribute.
If this AdditionalAttribute record is associated to an Attribute record and the flag IsAllowedValueDefined is set to 'Y', then either Value or IntegerValue or DoubleValue must be one of the computed values from the AttributeAllowedValue record.
YSC84_0006Invalid Value Assigned To The Attribute.
