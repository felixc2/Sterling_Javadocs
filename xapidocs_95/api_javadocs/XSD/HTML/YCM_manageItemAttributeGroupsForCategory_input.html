ItemAttributeGroupSequence
The ItemAttributeGroupSequence element defines a SequenceNo for a category and group combination. It is possible to define a sequence number for a group that does not belong to this category. In that case, if ItemAttributeGroupID is passed instead of ItemAttributeGroupKey, either CategoryPath or both CategoryID and ParentCategoryKey should be passed to identify the category that is associated with the ItemAtributeGroupID.
Note that only ItemAttributeGroupID or ItemAttributeGroupKey is required. Category-related attributes are only required if ItemAttributeGroupID is passed and the group is not owned by the category being modified. In all cases, CategoryKey and ItemAttributeGroupType will be defaulted.

Validation ConditionError
CategoryID, ParentCategoryKey, or CategoryPath are passed but do not identify a category.
YCM0034Category not found.
Attributes to uniquely identify a ItemAttributeGroup are not passed.
