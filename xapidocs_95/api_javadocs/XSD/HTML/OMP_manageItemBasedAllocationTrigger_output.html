ElementsItemBasedAllocationTrigger
ItemBasedAllocationTrigger Associated With Table: YFS_IBA_TRIGGER
This table stores information for inventory item and node, which were triggered to be processed by the Item Based Allocation agent.
AttributeTypeRequiredTable.Column
IBARequiredflag (maximum length: 1) yfs_iba_trigger.iba_required
This flag indicates that Item Based Allocation process is required for this item and node combination. Valid Values: Y/N.
IBARunRequiredflag (maximum length: 1) yfs_iba_trigger.iba_run_required
This flag indicates that Item Based Allocation agent must process this trigger. This will not be set to Y unless the IBA_REQUIRED field is set to Y. Valid Values: Y/N.
IBATriggerKeykey (maximum length: 24) yfs_iba_trigger.iba_trigger_key
The primary key attribute of the YFS_IBA_TRIGGER table.
ItemIDstring (maximum length: 40) Yesyfs_iba_trigger.item_id
The identifier of the inventory item.
LastIBAProcessedTimestampdateTimeyfs_iba_trigger.last_iba_processed_ts
The timestamp of the last time that the Item Based Allocation agent successfully processed this trigger.
Nodekey (maximum length: 24) Yesyfs_iba_trigger.node_key
The node that is triggered to be processed by the Item Based Allocation agent.
OrganizationCodestring (maximum length: 24) Yesyfs_iba_trigger.organization_code
The inventory organization that maintains the inventory item.
ProcessOverByTimestampdateTimeyfs_iba_trigger.process_over_by_ts
The timestamp that system expects the Item Based Allocation process to be over by. System will treat the PROCESSING_BY_AGENT flag as N regardless of the actual value when current timestamp is after this timestamp. This can be used in the scenario when the Item Based Allocation agent does not complete its operation due to some failures and the PROCESSING_BY_AGENT flag does not get reset to N. Default Value: current timestamp.
ProcessingByAgentflag (maximum length: 1) yfs_iba_trigger.processing_by_agent
This flag indicates that this trigger is being processed by the Item Based Allocation agent.
ProductClassstring (maximum length: 10) yfs_iba_trigger.product_class
The product class of the inventory item.
UnitOfMeasurestring (maximum length: 40) yfs_iba_trigger.uom
The unit of measure of the inventory item.
