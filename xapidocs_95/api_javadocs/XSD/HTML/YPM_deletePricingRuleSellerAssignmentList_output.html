PricingRule Associated With Table: YPM_PRICING_RULE
This table provides the information for each pricing rule.
AttributeTypeRequiredTable.Column
CarrierServiceCodestring (maximum length: 40) ypm_pricing_rule.carrier_service_code
The carrier service that this rule applies on. Applicable only for shipping category of rules.
Currencystring (maximum length: 20) ypm_pricing_rule.currency
Currency in which this rule is applicable.
DeleteAllCustomerAssignmentsflag (maximum length: 1) .
Valid Values are Y and N. If Y is passed then api deletes all customer assignments for the given pricing rule. If enterprise code is not passed then no customer assignment gets deleted.

Descriptionstring (maximum length: 2000) ypm_pricing_rule.description
Description of the rule.
DistributeAdjustmentflag (maximum length: 1) ypm_pricing_rule.distribute_adjustment
Boolean indicating if the rule adjustments should be distributed to lines or not.
EndDateActivedateTimeypm_pricing_rule.end_date_active
After this date the rule is not active.
IgnoreRuleOnManualAdjustmentflag (maximum length: 1) ypm_pricing_rule.ignore_rule_on_manual_adj
Indicates that this rule is not applicable on entire order or lines that have either manual adjustments or manual rule adjustments specified.
IgnoreRuleOnPriceLockflag (maximum length: 1) ypm_pricing_rule.ignore_rule_on_price_lock
If set indicates that the rule should not be fired if the line item's unit price is locked. The line item can be the target or the rule trigger. Applicable only for rules that are item targeted, like combination and item quantity rules.
IsCouponRuleflag (maximum length: 1) ypm_pricing_rule.is_coupon_rule
Indicates whether this rule is a coupon or not.
IsCustomRuleflag (maximum length: 1) ypm_pricing_rule.is_custom_rule
Indicates whether the rule is custom or not.
IsItemAttributeValueRuleflag (maximum length: 1) ypm_pricing_rule.is_item_attribute_value_rule
Indicates that this item targeted rule is defined on item attribute values. Applicable for all types of rules.
IsItemRuleflag (maximum length: 1) ypm_pricing_rule.is_item_rule
Indicates whether the rule is targeted on items or not.
IsManualRuleflag (maximum length: 1) ypm_pricing_rule.is_manual_rule
Indicates whether this pricing rule is a manually applied adjustment rule or not. A manual pricing rule is suggested by the pricing service based on contents or total of the order, which then again should be specified with the amount of adjustment to be applied against the order.
ItemOperatorCodestring (maximum length: 40) ypm_pricing_rule.item_operator_code
Indicates how the trigger for the rule is evaluated. Applicable only for combination type of rule.
Allowed ValueDescription
ALL_ITEMS Indicates that the rule condition is satisfied if all of the items in the condition is present. 
ANY_ITEM Indicates that the rule condition is satisfied if any of the item in the condition is present. 

MinimizeNumberOfShipmentsflag (maximum length: 1) ypm_pricing_rule.minimize_number_of_shipments
Indicates whether to optimize on the number of shipments or not. Applicable only for shipping category of rules.
OrganizationCodestring (maximum length: 24) ypm_pricing_rule.organization_code
Pricing organization to which the configuration belongs.
PricingRuleKeykey (maximum length: 24) ypm_pricing_rule.pricing_rule_key
Primary key attribute of YPM_PRICING_RULE table.
PricingRuleNamestring (maximum length: 120) ypm_pricing_rule.pricing_rule_name
Name of the pricing rule.
PricingStatusstring (maximum length: 40) ypm_pricing_rule.pricing_status
Indicates the status of the pricing rule.
Allowed ValueDescription
ACTIVE Indicates that pricing entity is active. 
INACTIVE Indicates that pricing entity is not active. 

RuleCategorystring (maximum length: 40) ypm_pricing_rule.rule_category
Indicates the category to which the rule belongs.
Allowed ValueDescription
PRICING Indicates that this rule belongs to pricing category. This rule generates adjustments for extended price of a line or for order total before shipping and tax. 
SHIPPING Indicates that this rule belongs to shipping category. This rule generates shipping surcharge for a line or adjustment on shipping charge of the order. 

RuleTypestring (maximum length: 40) ypm_pricing_rule.rule_type
Indicates the type of the rule.
Allowed ValueDescription
COMBINATION Indicates that this rule is applied based on combination of items in the order. 
ORDER_TOTAL Indicates that this rule is applied based on order total before shipping or tax. 
ITEM_QUANTITY Indicates that this rule is applied based on item and it's quantity in the order. 
SURCHARGE Indicates that the rule is for applying a surcharge. 
GLOBAL_EXCLUSION Indicates that the rule is for excluding certain items, categories or attributes from all pricing rules. 

SellerOrganizationCodestring (maximum length: 24) ypm_pricing_rule.seller_organization_code
Seller who maintains this pricing rule. Promotion is defined by the seller.
StartDateActivedateTimeypm_pricing_rule.start_date_active
Date from which the rule is active.
