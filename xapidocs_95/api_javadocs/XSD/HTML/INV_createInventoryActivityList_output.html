InventoryActivity
InventoryActivity Associated With Table: YFS_INVENTORY_ACTIVITY
This table stores inventory items whose supply or demand has been adjusted. The RealTimeAvailabilityMonitor agent will process the inventory item with inventory activities while running in the ActivityBased mode, and remove the processed records from this table.

ProcessedFlagflag (maximum length: 1) yfs_inventory_activity.processed_flag
The flag indicates whether this inventory activity has been picked up for processing by the Real-Time Availability Monitor. Valid values: [F,N] Default: N
Specify:
F for Force Publish. Indicating that inventory activity has not been processed and the Real-Time Availability Monitor should publish the availability picture regardless whether it has changed from the previously published availability.
N otherwise.
