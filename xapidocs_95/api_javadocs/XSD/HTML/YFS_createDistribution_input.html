ElementsItemShipNode
ItemShipNode Associated With Table: YFS_ITEM_SHIP_NODE
Stores distribution setup for an organization. This setup is used during any inventory operation.
If ItemType is passed as ALL , the ItemId has to be set to ALL .The calling program should validate this before calling the API.
The key to looking up a distribution rule is the OwnerKey and DistributionRuleID.

Validation ConditionError
A record is not found in YFS_DISTRIBUTION_RULE table for the input OwnerKey and DistributionRuleId pair.
YFS10001YFS:No Record Found.
