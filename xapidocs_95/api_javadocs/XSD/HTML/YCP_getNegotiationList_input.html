Negotiation Associated With Table: YFS_NEGOTIATION_HDR
Primary negotiation table. This table stores all order negotiations in the system. There can be zero or many negotations for an order.
AttributeTypeRequiredTable.Column
ApplicationActionstring (maximum length: 40) yfs_negotiation_hdr.application_action
Not used.
ApplicationReferencestring (maximum length: 40) yfs_negotiation_hdr.application_reference
Not used.
DocumentNostring (maximum length: 40) yfs_negotiation_hdr.document_no
Order number of the order for this negotiation is created.
DocumentTypestring (maximum length: 40) yfs_negotiation_hdr.document_type
Document type of the order for this negotiation is created.
EnterpriseKeykey (maximum length: 24) yfs_negotiation_hdr.enterprise_key
Enterprise of the order for which this negotiation is created.
This attribute corresponds to the OrganizationCode Fact.
FreightTermsstring (maximum length: 40) yfs_negotiation_hdr.freight_terms
This is FOB of order. This can be configured as a negotiable attribute in yfs_negotiable_hdr_attr table.
HeaderEntitystring (maximum length: 40) yfs_negotiation_hdr.header_entity
This is base document type of negotiation. Currently, it can have value ORDER only.
HeaderEntityKeykey (maximum length: 24) yfs_negotiation_hdr.header_entity_key
Primary key of the order for which this negotiation is created.
HeaderTermsNegotiatedflag (maximum length: 1) yfs_negotiation_hdr.header_terms_negotiated
Flag to indicated if all negotiation on all header level terms is completed or not.
InitiatorOrgCodestring (maximum length: 24) yfs_negotiation_hdr.initiator_org_code
This is order's buyer's organization code.
MaximumRecordsinteger (total number of digits: 9) .
Specifies the Maximum Number of records to be returned. Default is 500.
NegotiationExpirationDatedateTimeyfs_negotiation_hdr.negotiation_expiration_date
Date when negotiation expires. No logic is built on this. But, negotiation monitor can be configured to raise alerts on this.
NegotiationHeaderKeykey (maximum length: 24) yfs_negotiation_hdr.negotiation_hdr_key
Primary key. Always system generated.
This attribute corresponds to the PrimaryKey Fact.
NegotiationNostring (maximum length: 40) yfs_negotiation_hdr.negotiation_no
Part of unique key of negotiation. Enterprise code and negotiation number together uniquely identify a negotiation.
NegotiatorOrgCodestring (maximum length: 24) yfs_negotiation_hdr.negotiator_org_code
This is order's seller organization code.
PaymentTermsstring (maximum length: 20) yfs_negotiation_hdr.payment_terms
This is terms_code of order. This can be configured as a negotiable attribute in yfs_negotiable_hdr_attr table.
ReadFromHistorystring (maximum length: 1) .
Default: N
Valid Values:
N - Returns records that satisfy the input criteria from the YFS_NEGOTIATION_HDR table.
Y - Returns records that satisfy the input criteria from the YFS_NEGOTIATION_HDR_H table (the history table).
B - Returns records that satisfy the input criteria from both YFS_NEGOTIATION_HDR and YFS_NEGOTIATION_HDR_H tables. Records returned from the current table are first in the output.

NegotiationLine Associated With Table: YFS_NEGOTIATION_LINE
This table stores line information of a negotiation. One negotiation line is created for each order line of the negotiation's order.
AttributeTypeRequiredTable.Column
ApplicationActionstring (maximum length: 40) yfs_negotiation_line.application_action
Action to be performed on the order if negotiated quantity is less than the original quanitity. Currently, only CANCEL is supported.
ApplicationReferencestring (maximum length: 40) yfs_negotiation_line.application_reference
Not used.
DeliveryDatedateTimeyfs_negotiation_line.delivery_date
This is request delivery date of the order line. Depending on the order driver date, either delivery_date or ship_date is negotiable.
FreightTermsstring (maximum length: 40) yfs_negotiation_line.freight_terms
This is FOB on order line to which this negotiation line corresponds to. This can be configured as a negotiable attributes in yfs_negotiation_line_attr table.

Currently can have value ORDER_LINE only.
LineEntityKeykey (maximum length: 24) yfs_negotiation_line.line_entity_key
Primary key of the order line to which this negotiation line corresponds too.
