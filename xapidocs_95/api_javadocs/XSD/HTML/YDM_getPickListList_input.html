PickList Associated With Table: YFS_PICK_LIST
Stores picklist created during the picklist creation process. A picklist is a document used for picking. It is generated for a ship node across one or more shipments and contains information on the shipment lines and the quantity for each line that is to be picked. Picklist is created by the printPickList API.
