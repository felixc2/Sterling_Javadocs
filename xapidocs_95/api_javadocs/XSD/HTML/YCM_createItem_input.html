AttributeTypeRequiredTable.Column
ComputeUnitcostflag (maximum length: 1) .
If this attribute is passed, Item Unit cost will be computed as follows, For Physical Kit Items Unit cost is computed as the total of all the component unit costs extended by the kit quantity PLUS the WIP cost factor of the physical kit item. If the WIP cost factor of the physical kit item is not defined, then the WIP cost factor for the catalog organization's primary enterprise is used. For cost factor calculation QUANTITY,WEIGHT and VOLUME attributes are used from parent physical kit item. For PERCENTAGE based calculation the unit cost computed as described above, will be used. For Logical Kit Items Unit cost is computed as the unit price of the primary supplier's default price program PLUS the primary supplier's standard cost factor. If the primary supplier's standard cost factor is not defined, the standard cost factor for the primary supplier's primary enterprise is used. For Regular Items Unit cost is computed as the unit price of the primary supplier's default price program PLUS the primary supplier's landed cost factor and the primary supplier's standard cost factor applied on that value. If the primary supplier's standard/landed cost factors are not defined, the standard/landed cost factors for the primary supplier's primary enterprise is used.
