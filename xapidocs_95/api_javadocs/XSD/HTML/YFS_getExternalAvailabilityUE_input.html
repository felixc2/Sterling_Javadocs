Nodestring (maximum length: 24) .
Shipping node from which the inventory should be found.

ShipNodes
Multiple shipnodes can be passed for promise header and promise lines/promise service lines.
For promise header, multiple shipnodes will be used when:
1. ShipNode attribute on promise line/promise service line is not provided.
2. Multiple shipnodes on promise line/promise service line are not provided.
3. ShipNode attribute on promise header is not provided.

For promise line/promise service line, multiple shipnodes will be used when:
1. ShipNode attribute on promise line/promise service line is not provided.

If multiple shipnodes are passed for a DS line then the shipnode/s on the associated product line are ignored.
