ShipmentTagSerial Associated With Table: YFS_SHIPMENT_TAG_SERIAL
Stores additional information related to tag number, serial number, ship by date for a shipment line or container detail. Tag information is stored only if the Node tracks tag information in inventory or the Buyer mandates tag information.
