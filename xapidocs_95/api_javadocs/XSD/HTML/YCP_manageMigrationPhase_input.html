AttributeTypeRequiredTable.Column
Operationstring (maximum length: 40) .
Operation to perform
Valid Values:
Manage - Creates a new record if it does not exist, or modifies an existing record if it does.
Create - Creates a new record.
Modify - Change values in an existing record.
Delete - Deletes an existing record.
The default behavior is Manage .

Phasestring (maximum length: 40) Yes.
The Phase to which to transition.
Allowed ValueDescription
INITIAL The initial Migration Phase. Migration has not started for existing records, and changes to RDBMS not will be propagated to Cassandra. 
BEGIN_PROPAGATION All future RDBMS changes will be propagated to Cassandra. 
BEGIN_MIGRATION Existing records will be migrated from RDBMS to Cassandra. Changes to RDBMS will continue to be propagated to Cassandra. 
BEGIN_VERIFICATION Some applicable transactions will begin to operate using data from Cassandra. 
COMPLETE_VERIFICATION Applicable transactions will continue to operate using data from Cassandra. Reading from and writing to RDBMS will be discontinued for applicable tables/transactions.
