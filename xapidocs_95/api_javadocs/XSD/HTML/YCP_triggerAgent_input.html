TriggerAgent
The Api takes either CriteriaId or a combination of BaseTransactionId and ProcessType as the input.The attribute CommitOnParent helps the caller to commit the transaction along with the parent transaction. If CriteriaId is passed as input to this api, each attribute of CriteriaAttributes element override the corresponding attribute configured for CriteriaId . If CriteriaId is not passed, the attributes passed in the CriteriaAttributes element are used to trigger the agent that corresponds to these criteria attributes.
