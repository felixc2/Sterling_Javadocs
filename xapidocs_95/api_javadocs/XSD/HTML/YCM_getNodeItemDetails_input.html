Item Associated With Table: YFS_ITEM
Pass one of the following to identify the Item:
-ItemKey
-OrganizationCode, ItemID, and either UnitOfMeasure or AlternateUOM
Blank UnitOfMeasure is a valid value. If UnitOfMeasure is not passed in the input XML, it will be considered as blank.
-OrganizationCode and GlobalItemID
Note: If OrganizationCode and CallingOrganizationCode are passed, CallingOrganizationCode will be ignored.

Validation ConditionError
If data passed does not include one of the combinations to identify the Item.
YCM0002Missing Mandatory Parmeters.
