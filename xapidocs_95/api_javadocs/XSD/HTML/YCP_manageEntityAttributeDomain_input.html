AttributeTypeRequiredTable.Column
AttributeDomainIDstring (maximum length: 40) yfs_entity_attribute_domain.attribute_domain_id
The AttributeDomainID with which this entity attribute domain should be associated. If not passed in, a new attribute domain with AttributeDomainID as EntityAttribute is created. If passed in and no matching attribute domain already exists, a new AttributeDomain with the provided AttributeDomainID is created. In both the cases, AttributeGroup is also created with the same AttributeGroupID as the AttributeDomainID.

Validation ConditionError
AttributeDomainID is modified for an attribute domain that has existing attributes.
YSC91A_010Cannot modify attribute domain with existing attributes

DocumentTypestring (maximum length: 40) Yesyfs_entity_attribute_domain.document_type
The DocumentType with which the AttributeDomain defined in column ATTRIBUTE_DOMAIN_ID must be associated.
Operationstring (maximum length: 40) .
Operation to perform
Valid Values:
Manage - Creates a new record if it does not exist, or modifies an existing record if it does.
Create - Creates a new record.
Modify - Change values in an existing record.
Delete - Deletes an existing record.
The default behavior is Manage .

OrganizationCodestring (maximum length: 24) Yesyfs_entity_attribute_domain.organization_code
The organization that maintains the YFS_ATTRIBUTE_DOMAIN record given in ATTRIBUTE_DOMAIN_ID column.
This attribute corresponds to the OrganizationCode Fact.
TableNamestring (maximum length: 40) Yesyfs_entity_attribute_domain.table_name
The TableName with which the AttributeDomain defined in column ATTRIBUTE_DOMAIN_ID must be associated. The allowed values are YFS_ORDER_HEADER, YFS_ORDER_LINE AND YFS_OPPORTUNITY
