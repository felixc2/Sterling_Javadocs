ElementsClassificationItemAttribute
ClassificationItemAttribute Associated With Table: YFS_CLASS_ITEM_ATTR
This table stores common item attributes, which can be derived to items belonging to the specified classification during runtime.
AttributeTypeRequiredTable.Column
ATPRulekey (maximum length: 24) yfs_class_item_attr.atp_rule
The available to promise rule to be used for the item to determine availability.

The advance notification time for this item. Communication to ship the order to shipping node/supplier is made at most this many days before the expected shipment date. Used during the release process.
AssumeInfiniteInventoryflag (maximum length: 1) yfs_class_item_attr.assume_infinite_inventory
This flag indicates whether an item is assumed to be always available.

OnhandSafetyFactorPercentage will be used only when OnhandSafetyFactorQuantity is not specified.
