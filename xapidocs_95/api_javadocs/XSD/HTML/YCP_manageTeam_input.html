CustomerAccessModestring (maximum length: 2) yfs_data_security_group.customer_access_mode
Access mode for Customer. Indicates if the team can access all customer data (01), only specified customers (02) assigned to the team, or only specified customers assigned directly to the user, regardless of the Enterprise's manual assignment rule (03). Default Value is 01.
Allowed ValueDescription
01 Normal access mode. The team can access all customer data. 
02 Restricted based on team. Access to customer data is restricted based on team assignments. 
03 Restricted based on user. Access to customer data is restricted based on direct customer assignments regardless of the Enterprise's manual assignment rule.
