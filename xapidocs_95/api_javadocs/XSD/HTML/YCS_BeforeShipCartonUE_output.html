CommodityRecord Associated With Table: YCS_COMMODITY_RECORD
This table stores item level information for an international UPS shipper. So there will be one record for each item of the carton in the manifest.
AttributeTypeRequiredTable.Column
AgreementTypestring (maximum length: 10) ycs_commodity_record.agreement_type
The types of trade agreements that are applicable for the commodities contained in the invoice line, as indicated on the commercial/customs invoice. EEC European Economic Community EFTA European Free Trade Agreement NAFTA North American Free Trade Agreement.
COTypestring.

CertOfOriginCodestring (maximum length: 5) ycs_commodity_record.cert_of_origin_code
The code representing the type of origin certification that applies to the commodity.
CertOfOriginNostring (maximum length: 10) ycs_commodity_record.cert_of_origin_no
The number found on Certificate Of Origin.
CommodityCodestring (maximum length: 30) ycs_commodity_record.commodity_code
The harmonized code representing the commodity as indicated on the commercail invoice.
CommodityRemarksstring (maximum length: 200) ycs_commodity_record.commodity_remarks
Commodity level remarks produced by the customer. Example: Commodity Level Declaration Statements.
ECCNstring (maximum length: 15) ycs_commodity_record.eccn
The Export Commodity Control Number for the commodity.
InvoiceLineNumberstring (maximum length: 3) ycs_commodity_record.invoice_line_number
The line number associated with the commodity or invoice line as indicated on the commercial invoice.
LineCurrencyCodestring (maximum length: 3) ycs_commodity_record.line_currency_code
The Currency Code for the value of the commodity contained in the invoice line as indicated on the commercial invoice.
LineExtendedAmtdecimal (minimum value: 0
total number of digits: 22
fraction digits: 4) 		ycs_commodity_record.line_extended_amt
The value of the commodity in the invoice line, as indicated on the commercial invoice. This equals the LINE_UNIT_AMT_PRICE multiplied by the LINE_QUANTITY.
LineLicenseExpDatedateycs_commodity_record.line_license_exp_date
The expiry date for the validated export license, as indicated on the Waybill and/or S.E.D. If LINE_LICENSE_INFO is an exception, this is set to spaces.
LineLicenseInfostring (maximum length: 35) ycs_commodity_record.line_license_info
The validated export license number for the commodity contained in the shipment, as indicated on the Waybill and/or S.E.D. . This field is currently used for US and PR origins only, as specified by UPS. Required if the SED_Code equals D or U and shipment origin is US or PR.
LineMerchDesc1string (maximum length: 35) ycs_commodity_record.line_merch_desc1
The detail description of the commodity in the invoice line, as indicated on the commercial/customs invoice.
LineMerchDesc2string (maximum length: 35) ycs_commodity_record.line_merch_desc2
Continuation of the detail description of the commodity in the invoice line, as indicated on the commercial/customs invoice.
LineMerchDesc3string (maximum length: 35) ycs_commodity_record.line_merch_desc3
Continuation of the detail description of the commodity in the invoice line, as indicated on the commercial/customs invoice.
LineOriginCountrystring (maximum length: 2) ycs_commodity_record.line_origin_country
The country of manufacture or production or growth of the article. Must coform to ISO/IATA country code.
LineQtyUOMstring.

LineQuantitydecimal (minimum value: 0
total number of digits: 22
fraction digits: 4) 		ycs_commodity_record.line_quantity
The quantity of the commodity in the invoice line as indicated on the commercial invoice.
LineUnitAmtPricedecimal (minimum value: 0
total number of digits: 22
fraction digits: 4) 		ycs_commodity_record.line_unit_amt_price
The per unit value of the commodity in the invoice line as indicated on the commercial invoice.
Marks_Numbersstring (maximum length: 35) ycs_commodity_record.marks_numbers
Any distinguishing marks or numbers that appear on the package. This field is valid only for US, PR, CA and VI shippers, as specified by UPS.
NumberOfPackagesPerCmmdtydecimal (minimum value: 0
total number of digits: 22
fraction digits: 4) 		ycs_commodity_record.number_of_packages_per_cmmdty
The number of packages in the shipment containing this commodity. This field is currently used for US and PR origins, as specified by UPS.
PartNumberstring (maximum length: 35) ycs_commodity_record.part_number
The shipper's part number or reference number for the commodity as indicated on the commercial invoice.
QuantityScheduleBUnitsstring.

InternationalRecord Associated With Table: YCS_INTERNATIONAL_RECORD
This table stores UPS international shipper information.
AttributeTypeRequiredTable.Column
BrokerCodestring (maximum length: 3) ycs_international_record.broker_code
Identifies the broker for the shipment.
COCodestring (maximum length: 1) ycs_international_record.cocode
This field indicates if Certificate of Origin was completed or not, and the form it was on. As per UPS specification, this field is currently used for US and PR origins only; valid values are Space(No Certificate of Origin created), D(US Certificate of Origin created through shipping system and included with export documents), N(NAFTA Certificate of Origin created through shipping system and included with export documents, U(US Certificate of Origin prepared by UPS on shippers behalf).
ConsolidatedClearQtydecimal (minimum value: 0
total number of digits: 22
fraction digits: 4) 		ycs_international_record.consolidated_clear_qty
The number of consignees in the consolidated clearance.
DescriptionOfGoodsstring (maximum length: 50) ycs_international_record.description_of_goods
A general description of the goods contained in the shipment.
DestinationControlstring (maximum length: 200) ycs_international_record.destination_control
The destination control statement,as per UPS specification.
DiscountRebatedecimal (minimum value: 0
total number of digits: 22
fraction digits: 4) 		ycs_international_record.discount_rebate
Discount or Rebate amount as indicated on the commercial invoice.
FreightChargesdecimal (minimum value: 0
total number of digits: 22
fraction digits: 4) 		ycs_international_record.freight_charges
The freight amount as indicated on the commercial invoice.
InsuranceChargesdecimal (minimum value: 0
total number of digits: 22
fraction digits: 4) 		ycs_international_record.insurance_charges
The Insurance Amount as indicated on the commercial invoice.
InvoiceCurrencyCodestring (maximum length: 3) ycs_international_record.invoice_currency_code
The currency code used for INVOICE_LINE_TOTALS. This must conform to IATA/ISO country codes.
InvoiceDatedateycs_international_record.invoice_date
The ship time stamp.
InvoiceLineTotalsdecimal (minimum value: 0
total number of digits: 22
fraction digits: 4) 		ycs_international_record.invoice_line_totals
The value of the goods determined by shipper, as indicated on the waybill or commercial invoice.This is the sum of all LINE_EXTENDED_AMT in YCS_COMMODITY_RECORD table for the shipment.
InvoiceNumberstring (maximum length: 35) ycs_international_record.invoice_number
The invoice number as indicated on the invoice/waybill.
InvoiceSubTotaldecimal (minimum value: 0
total number of digits: 22
fraction digits: 4) 		ycs_international_record.invoice_sub_total
This is the result of subtracting DISCOUNT_REBATE from INVOICE_LINE_TOTALS.
OtherChargesdecimal (minimum value: 0
total number of digits: 22
fraction digits: 4) 		ycs_international_record.other_charges
Sum of the other amounts being charged as indicated on the commercial invoice.
OtherDocCodestring (maximum length: 1) ycs_international_record.other_doc_code
This indicates if any additional document is attached to the shipper. As per UPS specification, valid values are 0(no documents attached), 1(documents are attached), space(not applicable).
PONumberstring.

PartiesToTransstring (maximum length: 1) ycs_international_record.parties_to_trans
If SED is applicable, it indicates whether shipper and consignee/importer are related or non-related.If not applicable, this is set to space. As per UPS specification, this is currently used for US and PR origins only; valid values are R(related), N(not related), space(not applicable).
PaymentTermsstring (maximum length: 35) ycs_international_record.payment_terms
Description of the payment terms.
ReasonForExportstring (maximum length: 35) ycs_international_record.reason_for_export
The reason the shipment is to be exported as indicated on the commercial invoice.
SEDCodestring.

ShipmentCommodityOriginstring (maximum length: 2) ycs_international_record.shipment_commodity_origin
The country code that identifies the country of origin for the commodities.
ShipmentInsuranceDeclaredValuedecimal (minimum value: 0
total number of digits: 22
fraction digits: 4) 		ycs_international_record.shipmentinsurancedeclaredvalue
The value for goods that is determined by shipper for insurance purpose.
ShipmentSEDCASNumstring.

SpecialInstructionsstring (maximum length: 69) ycs_international_record.special_instructions
Shipper's comments or special instructions.
TermsOfShipmentstring (maximum length: 3) ycs_international_record.terms_of_shipment
Terms of delivery as indicated on the commercial invoice. As per UPS specification, valid values are CFR(Cost and Freight), CIF(Cost Insurance and Freight), CIP(Carriage and Insurance Paid), CPT(Carriage Paid To), DAF(Delivered At Frontier), DDP(Delivered Duty Paid), DDU(Delivered Duty Unpaid), DEQ(Delivered Ex Quay), DES(Delivered Ex Ship), EXW(Ex Works), FAS(Free Along Side), FCA(Free Carrier), FOB(Free on Board), Space(Not applicable).
TotalInvoiceAmountdecimal (minimum value: 0
total number of digits: 22
fraction digits: 4) 		ycs_international_record.total_invoice_amount
The sum of INVOICE_LINE_TOTALS, DISCOUNT_REBATE, FREIGHT_CHARGES, INSURANCE_CHARGES and OTHER_CHARGES.
UltimateDestCountrystring (maximum length: 2) ycs_international_record.ultimate_dest_country
The country of ultimate destination, as indicated on the waybill or SED. As per UPS specification, this is required when an Ultimate Consignee address is completed on the S.E.D.
WaybillNumber_BrokerageIDstring (maximum length: 11) ycs_international_record.waybillnumber_brokerageid
The waybill/brokerage Identification Number.
WaybillPrintIndstring (maximum length: 1) ycs_international_record.waybill_print_ind
Indicates if Waybill was produced for the shipment. As per UPS specification, valid values are 1 (Waybill was produced for shipment), default value is 0(No waybill used for shipment).
