ResourcePoolCapacity Associated With Table: YFS_RES_POOL
This table stores information for a resource pool. A resource pool represents a collection of resources for providing services. The resources represented by a resource pool should have common characteristics (like regions serviced and skill sets). Capacity for providing services is maintained at the Resource Pool level.
AttributeTypeRequiredTable.Column
CapacityOrganizationCodekey (maximum length: 24) yfs_res_pool.capacity_organization_code
The code of the catalog organization. A resource pool is always created wihin the context of a capacity organization. An organization can source services from any resource pool within the same capacity organization.
This attribute corresponds to the OrganizationCode Fact.
