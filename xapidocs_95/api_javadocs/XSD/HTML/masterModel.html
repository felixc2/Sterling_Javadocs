Elements and attributes are described here, with an associated table and column name. These associations should be loosely applied and do not indicate that the element or attribute is associated only with the indicated table and column. Do not access tables directly. Model
A model is a terminal node and represents a configurable product or a reusable building block.
AttributeTypeRequiredTable.Column
Countrystring (maximum length: 120) .
The country identifier for the model.

Languagestring (maximum length: 120) .
The language identifier for the model.

Namestring (maximum length: 120) .
The human-readable model name.

Pathstring (maximum length: 480) .
The configurator path to a model. This path is 'mangled' such that special characters are replaced by their Unicode code preceded by an underscore char.

Typestring (maximum length: 120) .
The type of model.
Valid Values: model or ocg or oig
