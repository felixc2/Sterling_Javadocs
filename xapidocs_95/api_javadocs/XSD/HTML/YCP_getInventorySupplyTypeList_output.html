TriggerItemBasedAllocationflag (maximum length: 1) yfs_inventory_supply_type.trigger_iba
This flag indicates whether the supply type can trigger the Item-Based Allocation process when supply changes occur. Valid values: Y/N. Default value: N. This is available to be configured only for non on-hand supply types.
UseDemandTypeForItemBasedAllocationflag (maximum length: 1) yfs_inventory_supply_type.use_demand_type_for_iba
This flag indicates that the demand types, which consider this supply type, should be considered for Item Based Allocation. Valid values: Y/N. Default value: N.
