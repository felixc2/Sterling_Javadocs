ShipmentSubLineNononNegativeInteger (total number of digits: 5) yfs_shipment_line.shipment_sub_line_no
This is a part of the unique key. ShipmentSubLineNo along with the ShipmentLineNo is used to identify the logical kit component shipment line. The value of this field is zero - for kit parent shipment lines and shipment lines with items that are not kit components, and greater than zero if the the shipment line corresponds to a logical kit component item.
