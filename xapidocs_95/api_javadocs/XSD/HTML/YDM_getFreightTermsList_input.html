TransportationPreferencestring (maximum length: 2) yfs_freight_terms.transportation_preference
Specifies the transportation preference sequence in which the routing guide/table should be read. For example, say the freight term assigns preferences in the order of 1.) Buyer, 2.) Seller, and 3.) Others. When the transportation preference is 1, the Buyer's preference is used first. When not available, the next one in sequence (in this example, the Seller) will be used. If the transportation preference is 3, available transportation will be picked without specific order.
