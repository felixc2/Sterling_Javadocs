ElementsShipment
Shipment Associated With Table: YFS_SHIPMENT
Pass ShipmentKey or pass ShipmentNo, ShipNode and SellerOrganizationCode to identify the shipment.
AttributeTypeRequiredTable.Column
CreateNewShipmentflag (maximum length: 1) .
CreateNewShipment can be passed as Y by the user exit to indicate that the application must create a new shipment as are no existing shipments to which one could consolidate with. Not passing Y would mean, that the Shipment selected by the user exit would be used to consolidate with.
PerformESPCheckflag (maximum length: 1) .
PerformESPCheck can be passed as Y by the user exit to indicate that the application must perform ESP checks when new shipments are not being created. Not passing Y would mean, that the Shipment selected by the user exit would be not go through ESP checks.
SellerOrganizationCodestring (maximum length: 24) yfs_shipment.seller_organization_code
The seller organization to which the shipment belongs.
ShipNodestring (maximum length: 24) yfs_shipment.shipnode_key
The ship node where the shipment was created.
ShipmentKeykey (maximum length: 24) yfs_shipment.shipment_key
The primary key for the YFS_SHIPMENT table.
ShipmentNostring (maximum length: 40) yfs_shipment.shipment_no
The shipment number. This is a part of the unique key or business key.
