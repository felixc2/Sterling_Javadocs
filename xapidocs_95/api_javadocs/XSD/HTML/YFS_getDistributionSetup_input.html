AttributeTypeRequiredTable.Column
ConsiderAllNodesflag (maximum length: 1) .
Indicates whether all ship nodes are considered in the search.
Default: N
Valid Values: Y or N
If ConsiderAllNodes is passed as Y , the availability check is done without applying any distribution rule filter, and the DistributionRuleId is ignored.
If ConsiderAllNodes is passed as N , the DistributionRuleId passed or the calling organization's default DistributionRuleId is used for availability calculations.

Validation ConditionError
If ConsiderAllNodes is passed as Y , ConsiderUnassignedDemand cannot be passed as Y or N .

ConsiderInventoryNodeControlflag (maximum length: 1) .
Indicates whether inventory node control data were used in the inventory reporting.
Valid Values: Y or N
Default: N

ConsiderUnassignedDemandflag (maximum length: 1) .
Indicates whether unassigned demand should be considered.
Default: N
Valid Values: Y or N
If ConsiderUnassignedDemand is passed as Y then unassigned demands (demands without ship nodes) are also used in availability calculations.
Unassigned demands are matched to supplies using the demand owner's default DistributionRuleId and not the calling organization's DistributionRuleId.
If ConsiderAllNodes is passed then ConsiderUnassignedDemand has no affect at all on availability calculations.

Validation ConditionError
Must be an existing Organization.
