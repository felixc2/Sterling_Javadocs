ItemAttributeGroupType Associated With Table: YFS_ITEM_ATTR_GROUP_TYPE
Identifies the category and the ItemAttributeGroupType to retrieve ItemAttributeGroups for.
A Category can be identified by passing CategoryKey, or by passing OrganizationCode along with either CategoryPath or both CategoryID and ParentCategoryKey.
