These associations should be loosely applied and do not indicate that the element or attribute is associated only with the indicated table and column. Do not access tables directly. AttributeTypeRequiredTable.Column
Page
Contains metadata for the returned page
AttributeTypeRequiredTable.Column
GeneratedOndateTime.
Time the page set was generated on.
IsFirstPagestring.

Allowed ValueDescription
Y This is the first page in the set. 
N Any other page. 

IsLastPagestring.

Allowed ValueDescription
Y This is the last page in the set. 
N This is either not the last page, or the underlying pagination strategy doesn't know whether it is the last page. 

IsValidPagestring.

Allowed ValueDescription
Y This is the first page or there is more than one record in the API output. 
N This is not the first page and no records are available. This may happen if an invalid page is requested, for example. 

PageNumberinteger (total number of digits: 9) .
The page number of this page. Set to -1 if the page number is unknown.
PageSetTokenstring.
Reference to the current set of pages. This should be used in future getPage invocations to get more data, and passed to the deletePageCache API when the data is no longer needed.
PageSizeinteger (total number of digits: 9) .
Number of records per page
StartRowNumberinteger (total number of digits: 9) .
Row number of the first record in the page. Set to -1 if the row number is unknown.
TotalNumberOfPagesinteger (total number of digits: 9) .
Total number of pages in the page set. If this is unknown and the root element of the API output contains a TotalNumberOfRecords attribute, then this will be derived from that number and the page size. Set to -1 if the total number of pages is unknown.
