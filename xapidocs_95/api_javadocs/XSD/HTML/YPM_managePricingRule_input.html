DiscountAllCombinationItemsflag (maximum length: 1) ypm_pricing_rule.discount_all_combo_items

When the flag is Y the target items will be grouped together as a combination (the same way triggers are treated) and the discount will be applied to all the items that make the combination. For example a rule like the following can be created: Buy 2 of A and 3 of B, get $10 off 3 C and 2 D. The $10 discount will be applied to 3 instances of C and 2 instances of D (making the total discount $50). The discount is applied to each target combination for each trigger combination or for each reference applied to the order.
