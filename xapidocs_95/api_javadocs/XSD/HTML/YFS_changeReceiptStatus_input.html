ReceiptStatusAudit Associated With Table: YFS_RECEIPT_STATUS_AUDIT
This table stores receipt status change audit information. Each time the receipt status changes because of a business process, an audit is posted along with the status, timestamp, reason code and text, user ID and program ID. All transactions in the receipt pipeline will have entries in this table upon completion of a transaction.
