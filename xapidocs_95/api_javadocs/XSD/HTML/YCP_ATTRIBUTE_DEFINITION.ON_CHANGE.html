Attribute Associated With Table: YFS_ATTRIBUTE
This table stores attribute definitions. Attributes represent characteristics of entities and can be assigned to an attribute group.
AttributeTypeRequiredTable.Column
AllowMultipleValuesflag (maximum length: 1) yfs_attribute.allow_multiple_values
This flag indicates whether multiple values are allowed to be assigned for this attribute. Default value: "N".
AllowedValueLengthpositiveInteger (total number of digits: 3) yfs_attribute.allowed_value_length
The maximum length allowed for a value of Data Type = 'TEXT'. If not passed, the maximum length allowed is 200.
AttributeDomainIDstring (maximum length: 40) yfs_attribute.attribute_domain_id
The unique identification of the attribute domain.
AttributeGroupIDstring (maximum length: 40) yfs_attribute.attribute_group_id
The unique identification of the attribute group.
AttributeIDstring (maximum length: 40) yfs_attribute.attribute_id
The unique identifier of the attribute.
AttributeKeykey (maximum length: 24) yfs_attribute.attribute_key
The primary key for the YFS_ATTRIBUTE table.
AttributePostFixstring (maximum length: 40) yfs_attribute.attribute_post_fix
The post fix to be applied to the value of this attribute.
DataTypestring (maximum length: 20) yfs_attribute.data_type
The data type allowed for the value of the attribute.
Allowed ValueDescription
DECIMAL Allowed Value is a Decimal. 
BOOLEAN Allowed Value is a Boolean. 
INTEGER Allowed Value is an Integer. 
TEXT Allowed Value is a Text. 
DATE Allowed Value is a Date. 

IsAllowedValueDefinedflag (maximum length: 1) yfs_attribute.is_allowed_value_defined
This flag indicates whether a set of allowed values are defined for this attribute. Default value: "N".
IsValueMandatoryflag (maximum length: 1) yfs_attribute.is_value_mandatory
This flag indicates whether this attribute requires a mandatory selection of value when assigned to an item. Default value: "N".
LongDescriptionstring (maximum length: 255) yfs_attribute.long_description
The detailed description of the attribute.
Operationstring (maximum length: 40) .
Operation to perform
Valid Values:
Manage - Creates a new record if it does not exist, or modifies an existing record if it does.
Create - Creates a new record.
Modify - Change values in an existing record.
Delete - Deletes an existing record.
The default behavior is Manage .

ShortDescriptionstring (maximum length: 100) yfs_attribute.short_description
The short description of the attribute.

The primary key for the YFS_ATTR_ALLOWED_VALUE table.

The attribute that the allowed value belongs to.
IsDefaultValueflag (maximum length: 1) yfs_attr_allowed_value.is_default_value
This flag indicates whether this value is the default value of the attribute. Default value: "N".
LongDescriptionstring (maximum length: 255) yfs_attr_allowed_value.long_description
The detailed description of the allowed value.
NumericStepdecimal (total number of digits: 14
fraction digits: 4) 		yfs_attr_allowed_value.numeric_step
The incremental step of the allowed value within the numeric value range. For instance, if the numeric value range is from 0 to 100 and the numeric step is 5, the allowed values are 0, 5, 10, 15, 20, 25, etc.
SequenceNononNegativeInteger (total number of digits: 5) yfs_attr_allowed_value.seq_no
The sequence of the allowed value within the attribute definition.
ShortDescriptionstring (maximum length: 100) yfs_attr_allowed_value.short_description
The short description of the allowed value.
Valuestring (maximum length: 200) yfs_attr_allowed_value.value
The allowed value of the attribute that a user is entitled to assign. For an attribute of Numeric or Decimal Datatype, this value may be a list of values and/or a range of values. Each value in the list may be separated by a ';'. A range must be specified by two values separated by a '-'.

AttributeReference Associated With Table: YFS_ATTRIBUTE_REFERENCE
This table stores references of predefined attribute definition to fields of YFS_ITEM or item alias name.
AttributeTypeRequiredTable.Column
AttributeKeykey (maximum length: 24) yfs_attribute_reference.attribute_key
The attribute that this reference field is assigned to.
AttributeReferenceKeykey (maximum length: 24) yfs_attribute_reference.attribute_reference_key
The primary key attribute of YFS_ATTRIBUTE_REFERENCE table.
ReferenceNamestring (maximum length: 40) yfs_attribute_reference.reference_name
The reference name that the predefined attribute represents. If the tablename is 'YFS_ITEM', then this field contains the XML name of the field that the attribute represents. If the tablename is 'YFS_ITEM_ALIAS', then this field contains the item alias name that the attribute represents.
TableNamestring (maximum length: 40) yfs_attribute_reference.table_name
The name of the database table that contains the field assigned to a predefined attribute definition. Valid values : 'YFS_ITEM' or 'YFS_ITEM_ALIAS'.
