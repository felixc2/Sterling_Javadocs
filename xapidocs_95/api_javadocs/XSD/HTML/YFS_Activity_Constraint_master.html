LocationStorageCodestring (maximum length: 40) yfs_location_preference.location_storage_code
Ths field is an attribute of the location that allows the warehouse to store items, which has the same storage profile as that of the location. For example, hazardous inflammable items need locations close to fire extinguishers. In this case, the locations will be marked as having a storage code, which is suitable for storing 'Inflammable item'. This ensures that only inflammable items get to these locations. A list of zones in the order of priority will be used for putaway after filtering out locations that honor the storage code.
