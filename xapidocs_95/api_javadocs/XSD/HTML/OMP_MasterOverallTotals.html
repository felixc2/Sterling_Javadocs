IsManualstring (maximum length: 1) yfs_header_charges.is_manual
Determines whether the charge was manually created or created automatically through pricing engine.
Allowed ValueDescription
Y Indicates that the charge was created manually. 
N Indicates that the charge was automatically generated through the pricing engine. 
M Indicates that the charge was automatically generated through the pricing engine due to a manual pricing rule being applied. 

IsRefundableflag (maximum length: 1) yfs_charge_category.is_refundable
Indicates if a charge is refundable during returns.
IsShippingChargeflag (maximum length: 1) .
Determines whether the charge is a shipping charge. This attribute value is system defined and cannot be overridden. The value for this attribute is set as Y only for Charge Name configured as CHARGE_NAME_FOR_SHIPPING in financial rule. For all other Charges this attribute is set to N.
