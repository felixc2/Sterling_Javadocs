LocationStorageCode Associated With Table: YFS_LOCATION_STORAGE_CODE
A location may be associated with a location storage code. A storage code indicates a storage classification of location. It is an attribute of the location that allows the warehouse to store items that have the same storage profile as that of the location. For example, hazardous inflammable items need locations close to fire extinguishers. In this case the locations will be marked as having a storage code which is suitable for storing inflammable item. This ensures that only inflammable items get to these locations.
