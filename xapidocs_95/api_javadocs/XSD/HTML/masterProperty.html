Elements and attributes are described here, with an associated table and column name. These associations should be loosely applied and do not indicate that the element or attribute is associated only with the indicated table and column. Do not access tables directly. The tables should only be accessed through exposed APIs and Business Intelligence. Accessing them directly may result in issues when applying software updates.
Elementsproperty
property
This element holds information about properties associated to item.
AttributeTypeRequiredTable.Column
namestring (maximum length: 480) .
The name of the property.

pathstring (maximum length: 120) .
The path to the item that contains the property.

typestring (maximum length: 120) .
The datatype of the property.
Valid Values: bigdecimal or int or List or long or double or String or date or unknown

valuestring (maximum length: 480) .
The value in string form for the property. List values are returned with each value separated by a carriage return.
