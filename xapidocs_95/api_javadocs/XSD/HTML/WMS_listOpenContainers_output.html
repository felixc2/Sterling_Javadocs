OpenContainer
OpenContainer Associated With Table: YFS_OPEN_CONTAINER
This table stores information about all open pallets available in the system.
AttributeTypeRequiredTable.Column
IsShippingContainerflag (maximum length: 1) yfs_open_container.is_shipping_container
This flag indicates whether the container is a shipping container or inventory container.
LPNNostring (maximum length: 40) yfs_open_container.lpn_no
The case or pallet License Plate Number. This field is a foreign key to the YFS_LPN_HDR table.
LPNTypestring (maximum length: 40) yfs_open_container.lpn_type
This field indicates the system defined values for the LPN type associated with the LPN number for which an open container exists. Currently, the system supports only CASE and PALLET LPN types.
