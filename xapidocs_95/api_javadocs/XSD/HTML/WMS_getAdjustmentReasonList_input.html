AdjustmentLocationSequence Associated With Table: YFS_ADJUSTMENT_LOCN_SEQ
This table stores the zone or location sequence for inventory adjustments. During inventory adjustments, these locations or zones are considered in the order of sequence. If zones are configured in the adjustment sequence, locations within the zone are considered in the order of pick sequence and location id. Inventory is deducted from non-virtual locations till a virtual location is reached, and the rest of the inventory is deducted from that location. The inventory adjustment is such that only the virtual location configured in the adjustment sequence goes negative.
