getAlertStatisticsForUser
AttributeTypeRequiredTable.Column
IncludeAlertsAssignedToUserflag (maximum length: 1) .
If this Attribute is Y,statistics will be shown for exceptions assigned to the user.
This should not be set Y if user is subscribed to no queues and ShowAllForNoQueues is Y.
IncludeAlertsInUsersDepartmentQueueflag (maximum length: 1) .
If this Attribute is Y,statistics will be shown for exceptions in the users department queues in addition to those in his queues.
IncludeAlertsOwnedByUsersOrgflag (maximum length: 1) .
If this Attribute is Y,statistics will be shown for exceptions owned by users organization.
This should not be set Y if user is subscribed to no queues and ShowAllForNoQueues is Y.
ShowAllForNoQueuesflag (maximum length: 1) .
Default for this attribute is always Y.
If this is N,statistics won't we shown for all alerts when user is not subscribed to any Queues.
