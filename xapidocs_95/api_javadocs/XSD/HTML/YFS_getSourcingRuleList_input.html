FutureInventoryWindowdecimal (total number of digits: 14
fraction digits: 4) 		yfs_sourcing_rule_dtl.future_inventory_window
Indicates window in which future inventory can be considered. Future inventory coming outside of this window, will not be considered during scheduling. The default value is null which means this window will not be considered. The window should be a valid positive Integer or -1 if only onhand inventory is to be used.
