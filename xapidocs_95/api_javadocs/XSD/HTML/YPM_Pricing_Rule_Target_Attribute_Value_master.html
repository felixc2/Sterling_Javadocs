PricingRuleTargetAttributeValue Associated With Table: YPM_PRCNG_RULE_TARG_ATTR_VALUE
This table maintains information about the item attribute values of the items on which the pricing rule adjustments are targeted. Targeting by item attribute is allowed only for item quantity and surcharge rule types of pricing category only.
AttributeTypeRequiredTable.Column
GroupIdinteger (total number of digits: 9) ypm_prcng_rule_targ_attr_value.group_id
The group id of the attribute value. All attributes with same group id are evaluated together.
IncludeOrExcludestring (maximum length: 40) ypm_prcng_rule_targ_attr_value.include_or_exclude
Indicates whether certain items should be included or excluded from adjustments depending on item's attribute values.
Allowed ValueDescription
INCLUDE Indicates that combination of item and quantity is used to trigger the pricing rule. 
EXCLUDE Indicates that combination of item and quantity is used to filter out the pricing rule.
