Validation Associated With Table: YFS_VALIDATION
This table stores the configuration for a validation
AttributeTypeRequiredTable.Column
IsValidflag (maximum length: 1) .
This attribute indicates whether credit card validation was successful. The client checks the value of this attribute. If it is Y, the credit card is considered as valid and the client does not perform any additional validation. If it is N, the credit card is considered as invalid, the client does not perform any additional validation and displays an error on the screen. If it is blank, it is assumed that the credit card was not validated and the client performs its out of the box validation.
Messagestring (maximum length: 200) .
This attribute will have error messages in case the validation fails.
