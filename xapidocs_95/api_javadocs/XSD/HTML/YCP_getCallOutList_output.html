CallOutNamestring (maximum length: 255) plt_callout.callout_name
The name of the CallOut.
IsInputModifiableflag (maximum length: 1) plt_callout.is_input_modifiable
This should be set to Y if the input passed to each CallOut unit can be modified; The default value of this field is N which means the input passed to each of the callout units cannot be modified. Any character other than Y will be considered as N
IsThreadedflag (maximum length: 1) plt_callout.is_threaded
This should be set to Y if each CallOut unit needs to be executed in a separate thread; The default value of this field is N which means the callout unit will be executed in the same thread. Any character other than Y will be considered as N
