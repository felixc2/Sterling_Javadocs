ResourcePool Associated With Table: YFS_RES_POOL
This default resource pool to which the service resource belongs. A service resource is always associated with a default resource pool. Either ResourcePoolKey or {ResourcePoolId and CapacityOrganizationCode} can be used to identify a Resource Pool.

Validation ConditionError
The Resource Pool must already be present in the system. If not present:
INV10014Resource Pool for given criteria not found

ServiceResource Associated With Table: YFS_SERVICE_RESOURCE
This element has information to uniquely identify a service resource. A service resource is used to define one or more people that work as a team, and is represented as a single entity that performs certain work. Pass either ServiceResourceKey or {ServiceResourceId and Node} to identify an existing service resource.

Validation ConditionError
The root node must be ServiceResource. If not:
INV10003Invalid Root Node
To idenfity a service resource uniquely pass either of ServiceResourceKey or {ServiceResourceId and Node}, else :
INV80_015Either ServiceResourceKey or ServiceResourceId and Node should be passed to uniquely identify a service resource.
The service resource is being created and ResourcePool element is not passed.
INV80_016Default resource pool not specified for the service resource.
AttributeTypeRequiredTable.Column
Nodekey (maximum length: 24) yfs_service_resource.node_key
The node that is associated with this resource.
This attribute corresponds to the OrganizationCode Fact.
Notesstring (maximum length: 4000) yfs_service_resource.notes
Additional information about the service resource.
Operationstring (maximum length: 40) .
Valid value is Delete . Set Operation = " Delete " to delete the service resource.

Validation ConditionError
If there is capacity consumption for the service resource and Operation is Delete :
INV80_018Deletion of service resource is not allowed as the service resource has capacity consumptions.

ServiceResourceIdstring (maximum length: 24) yfs_service_resource.service_resource_id
Service Resource Identifier. It needs to unique within a node for the API to function.

Validation ConditionError
If a service resource already exists with same ServiceResourceId, Node combination and a service resource is getting created in this API call then:
INV80_024The service resource id with which the service resource is being created already exists in the node.
If there exists more than one service resource for the passed ServiceResourceId, Node combination:
INV80_025Service resource identifier is not unique within the node.

ServiceResourceKeykey (maximum length: 24) yfs_service_resource.service_resource_key
Primary key attribute of YFS_SERVICE_RESOURCE table.
This attribute corresponds to the PrimaryKey Fact.
ServiceResourceCalendar Associated With Table: YFS_CALENDAR
This element has information related to a calendar. A calendar can be created for an organization or for a ship node. A calendar is created using the createCalendar API and managed using the changeCalendar API.

Validation ConditionError
If provided, the calendar should be available and must belong to the node of service resource or its primary enterprise. Either CalendarKey or {OrganizationCode and CalendarId} must be passed.
INV10001Invalid Calendar
INV10032Node or its primary enterprise does not own the Calendar
