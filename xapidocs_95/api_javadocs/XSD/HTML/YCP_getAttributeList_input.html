The organization that has defined the attributes. If CallingOrganizationCode is not passed in the input, OrganizationCode is obtained from the logged in user. This attribute corresponds to the OrganizationCode Fact.
Purposestring (maximum length: 40) yfs_attribute_domain.purpose
The Purpose of the AttributeDomain to which this attribute belongs. If not passed in the input, a default value of 'ITEM' is used to fetch the atribute list.

ENTITY_EXTN Indicates that the attribute domain is intended for entity custom attributes.

AttributeReference Associated With Table: YFS_ATTRIBUTE_REFERENCE
This table stores references of predefined attribute definition to fields of YFS_ITEM or item alias name.
AttributeTypeRequiredTable.Column
AttributeKeykey (maximum length: 24) yfs_attribute_reference.attribute_key
The attribute that this reference field is assigned to.
AttributeReferenceKeykey (maximum length: 24) yfs_attribute_reference.attribute_reference_key
The primary key attribute of YFS_ATTRIBUTE_REFERENCE table.
ReferenceNamestring (maximum length: 40) yfs_attribute_reference.reference_name
The reference name that the predefined attribute represents. If the tablename is 'YFS_ITEM', then this field contains the XML name of the field that the attribute represents. If the tablename is 'YFS_ITEM_ALIAS', then this field contains the item alias name that the attribute represents.

The name of the database table that contains the field assigned to a predefined attribute definition. Valid values : 'YFS_ITEM' or 'YFS_ITEM_ALIAS'.
