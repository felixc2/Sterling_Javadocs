ResourcePool Associated With Table: YFS_RES_POOL
This table stores information for a resource pool. A resource pool represents a collection of resources for providing services. The resources represented by a resource pool should have common characteristics (like regions serviced and skill sets). Capacity for providing services is maintained at the Resource Pool level.

ServiceResource Associated With Table: YFS_SERVICE_RESOURCE
Provide criteria for service resource and/or resource pool to obtain service resource capacity information.

Validation ConditionError
If StartDate and EndDate are passed, StartDate must be before EndDate. If StartDate is after EndDate:
INV10026INV: StartDate must be before EndDate
AttributeTypeRequiredTable.Column
EndDatedate.
The end date of the date range. If EndDate is not passed, it defaults as follows:
if NumberOfDaysFromStartDate is passed, EndDate = StartDate + NumberOfDaysFromStartDate (including the Start Date)
else EndDate = StartDate + 7 (including the Start Date).

NumberOfDaysFromStartDateinteger (total number of digits: 9) .
The number of days (including start date) from the start date for the date range to the end date.
If NumberOfDaysFromStartDate is passed, it is used to calculate EndDate, if EndDate is not passed.
If NumberOfDaysFromStartDate is not passed, it defaults to 7 days.
