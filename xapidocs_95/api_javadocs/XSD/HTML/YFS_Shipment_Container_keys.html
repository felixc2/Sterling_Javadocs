Container Associated With Table: YFS_SHIPMENT_CONTAINER
Stores container header information. Containers are created using packShipment, createShipment or changeShipment API.
AttributeTypeRequiredTable.Column
ContainerNostring (maximum length: 40) yfs_shipment_container.container_no
The container number. This is a unique key or business key, which by itself identifies a delivery plan. If not passed as input, the system generates this number using the database sequence SEQ_YFS_CONTAINER_NO.
