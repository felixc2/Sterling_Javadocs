These associations should be loosely applied and do not indicate that the element or attribute is associated only with the indicated table and column. Do not access tables directly. ElementsEntityCustomAttribute
EntityCustomAttribute Associated With Table: YFS_ENTITY_CUSTOM_ATTRIBUTE

Validation ConditionError
CallingOrganizationCode or OrganizationCode must be passed.
YSC91A_006OrganizationCode Or CallingOrganizationCode Is Mandatory For This Operation.
AttributeTypeRequiredTable.Column
AttributeDomainIDstring (maximum length: 40) yfs_entity_custom_attribute.attribute_domain_id
The unique identifier of the attribute domain.
AttributeGroupIDstring (maximum length: 40) yfs_entity_custom_attribute.attribute_group_id
The unique identifier of the attribute group.
AttributeIDstring (maximum length: 40) yfs_entity_custom_attribute.attribute_id
The unique identifier of the attribute. This attribute will be used as an extended column for the TableName/DocumentType defined in the corresponding YFS_ENTITY_ATTRIBUTE_DOMAIN table.
CallingOrganizationCodestring (maximum length: 24) yfs_organization.organization_code
If CallingOrganizationCode is passed, the EntityCustomAttribute list for the organization that has configured custom attributes for this CallingOrganizationCode is returned.

This attribute corresponds to the OrganizationCode Fact.
ColumnNamestring (maximum length: 40) yfs_entity_custom_attribute.column_name
This points to the Column in the hang-off of the corresponding Entity for storing the extended attribute defined in ATTRIBUTE_ID field.
DocumentTypestring (maximum length: 40) yfs_entity_attribute_domain.document_type
The DocumentType with which the AttributeDomain defined in column ATTRIBUTE_DOMAIN_ID must be associated.
EntityAttributeDomainKeykey (maximum length: 24) yfs_entity_custom_attribute.entity_attribute_domain_key
The primary key for the YFS_ENTITY_ATTRIBUTE_DOMAIN table.
EntityCustomAttributeKeykey (maximum length: 24) yfs_entity_custom_attribute.entity_custom_attribute_key
The primary key for the YFS_ENTITY_CUSTOM_ATTRIBUTE table.
OrganizationCodestring (maximum length: 24) yfs_entity_custom_attribute.organization_code
The organization that has defined the custom attributes. If CallingOrganizationCode is not passed in the input, then this OrganizationCode passed in the input is used to fetch the list.

This attribute corresponds to the OrganizationCode Fact.
Searchableflag (maximum length: 1) yfs_entity_custom_attribute.searchable
This flag indicates whether the extended attribute is an indexed column in the hang-off. Default value: "N".
Statusstring (maximum length: 40) yfs_entity_custom_attribute.status
This represents the Status of the extended attribute.
Allowed ValueDescription
PENDING Indicates that the extended Attribute is mapped to a column in the hang-off of the Entity in TableName for the document type in DocumentType column. 
ACTIVE Indicates that the attribute can be used for storing transaction data. 
INACTIVE Indicates that the mapped column is not currently used. 
DELETED Indicates that the corresponding Column can now be re-mapped to another Attribute. 

TableNamestring (maximum length: 40) yfs_entity_attribute_domain.table_name
The TableName with which the AttributeDomain defined in column ATTRIBUTE_DOMAIN_ID must be associated. The allowed values are YFS_ORDER_HEADER, YFS_ORDER_LINE AND YFS_OPPORTUNITY
