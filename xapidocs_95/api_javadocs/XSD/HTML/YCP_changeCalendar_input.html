Calendar Associated With Table: YFS_CALENDAR
This is a mandatory element. It stores information related to a Calendar to be changed. A calendar can be created for an organization or for a ship node. A calendar is created by the createCalendar API and managed by changeCalendar API.

Validation ConditionError
Either CalendarKey or {CalendarId and OrganizationCode} must be passed. If not passed:
YFS10515Either CalendarKey or CalendarId and OrganizationCode must be passed.
Invalid Calendar information provided.
YFS10516Calendar not found.

EffectivePeriod Associated With Table: YFS_CALENDAR_SHIFT_EFF_PERD
This is an optional element. Stores information about the Shift Effective Periods for a Calendar. Shift Effective Periods can be created for a date range. Shift Effective Periods can be created or managed by the createCalendar API or changeCalendar API. A calendar can have multiple Effective Periods specified. However, a calendar that inherits from another calendar cannot define its own Effective Periods as they will be derived from the parent calendar.

Validation ConditionError
One EffectivePeriod should not overlap another. If they overlap:
YFS10511EffectivePeriods should not overlap
If this calendar is inheriting its definition from another calendar and an EffectivePeriod is specified:
YCP75_00007Inherited calendar cannot define any EffectivePeriods.

Shift Associated With Table: YFS_CALENDAR_SHIFT
This is an optional element. It stores information related to a shift. Shifts can be defined for a given effective period; this API indicates the ShiftName, ShiftStartTime, ShiftEndTime, and the day of week that shift is valid. A shift is created or managed by the createCalendar API or changeCalendar API.

Validation ConditionError
Shift must belong to the EffectivePeriod that CalendarDayException falls into. If not valid:
YCP0176Shift must belong to Effective Period that CalendarDayException falls into
ShiftStartTime and ShiftEndTime must be passed and must be valid. If not passed:
YFS10518ShiftStartTime and ShiftEndTime must be passed
Valid values for *Valid are Y or N . There is no validation for *Valid.
