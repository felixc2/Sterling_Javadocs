EffectiveFromDatedateyfs_node_transfer_schedule.effective_from_date
The date in which the specified transfer schedule goes into effect. An empty value means that this transfer schedule will be effective for any date up until the EffectiveToDate. In order to change this date, the NodeTransferScheduleKey must be passed along with the new date.

Validation ConditionError
The EffectiveFromDate overlaps an existing node transfer schedule.
OMP80_0024Can not create a transfer schedule with effective dates that overlap another transfer schedule for the same nodes.

EffectiveToDatedateyfs_node_transfer_schedule.effective_to_date
The last date in which the specified transfer schedule will be in effect. An empty value means that this transfer schedule will be effective for any date in the future. In order to change this date, the NodeTransferScheduleKey must be passed along with the new date.

Validation ConditionError
The EffectiveToDate overlaps an existing node transfer schedule.
OMP80_0024Can not create a transfer schedule with effective dates that overlap another transfer schedule for the same nodes.
The EffectiveToDate passed is before the EffectiveFromDate.
OMP80_0025Can not create a transfer schedule with an EffectiveToDate before EffectiveFromDate.
