ElementsCalendar
Calendar Associated With Table: YFS_CALENDAR
This element stores information related to a Calendar and is mandatory. Calendar can be created for an organization or for a ship node. Calendar is created by the createCalendar API and managed by the changeCalendar API.

Validation ConditionError
Either CalendarKey or both (CalendarID and OrganizationCode) must be passed. If not passed:
YFS10515CalendarKey or (CalendarID and OrganizationCode) must be passed
A calendar must be found that meets the given CalendarKey or (CalendarID and OrganizationCode) criteria. If no calendar is found:
YCP0102Calendar not found
FromDate and ToDate must be passed. If not passed:
YFS10523FromDate and ToDate are mandatory
FromDate must be earlier than ToDate. If ToDate is earlier than FromDate:
YFS10524FromDate must come before ToDate
A calendar must be found that meets the given FromDate and ToDate criteria. If no calendar is found:
YFS10516Calendar not found
