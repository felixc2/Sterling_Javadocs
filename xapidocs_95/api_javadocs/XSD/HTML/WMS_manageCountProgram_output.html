The primary key for the YFS_COUNT_PROGRAM table.

OwnerKeykey (maximum length: 24) yfs_count_program.owner_key

RequestTypestring (maximum length: 40) yfs_count_program.request_type
This field indicates the count request type.
RequestingUserIdstring (maximum length: 50) yfs_count_program.requesting_user_id
The user who requested for the count program.
StartNoEarlierThandateTimeyfs_count_program.start_no_earlier_than

Statusstring (maximum length: 15) yfs_count_program.status
The current status of the count program. The status is applicable only when the count program type is 'OneTime'.
